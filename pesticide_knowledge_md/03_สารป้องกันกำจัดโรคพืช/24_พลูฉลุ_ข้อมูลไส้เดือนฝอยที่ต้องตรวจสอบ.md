# คำแนะนำจากเอกสารสำหรับไส้เดือนฝอยในพลูฉลุ

## ข้อมูลที่ต้นฉบับระบุ

ต้นฉบับใช้ชื่อศัตรูว่า “โรครากปม” และระบุสาเหตุจากไส้เดือนฝอย _Radopholus similis_ จึงควรตรวจสอบชื่อโรคและชนิดไส้เดือนฝอยก่อนใช้จริง

## สารที่แนะนำ

| ชื่อสามัญ           | สูตร    | กลุ่ม IRAC | ระดับพิษและ LD50 | อัตราใช้   | วิธีใช้            |
| ------------------- | ------- | ---------- | ---------------- | ---------- | ------------------ |
| ไดโนทีฟูแรน         | 1% GR   | 4A         | น้อย (>2,000)    | 4 กรัม/ต้น | รองก้นหลุมก่อนปลูก |
| ฟิโพรนิล            | 0.3% GR | 2B         | ปานกลาง (92)     | 6 กรัม/ต้น | รองก้นหลุมก่อนปลูก |
| คาร์แทปไฮโดรคลอไรด์ | 4% GR   | 14         | ปานกลาง (250)    | 6 กรัม/ต้น | รองก้นหลุมก่อนปลูก |

ต้นฉบับหมายเหตุว่าสารทั้งสามเป็นสารกำจัดแมลง

## ประเด็นที่ต้องตรวจสอบ

1. ตรวจชนิดไส้เดือนฝอยจากรากและวัสดุปลูก
2. ตรวจชื่อโรคที่ถูกต้อง
3. ตรวจทะเบียนในพลูฉลุหรือไม้ประดับเป้าหมาย
4. ตรวจว่าฉลากอนุญาตให้ใช้กับไส้เดือนฝอยดังกล่าว
5. สูตร GR ใช้รองก้นหลุม ไม่ใช่ผสมน้ำพ่น

| -------- | --- | ---------- | ----------- | --------- | ----------------- | ---------- |
|          |     |            |             |           |                   |            |

## จุดที่ต้องระวัง

- อัตราเป็นกรัมต่อต้น ไม่ใช่กรัมต่อน้ำ
- ข้อมูลชื่อโรคกับชนิดไส้เดือนฝอยต้องตรวจสอบเพิ่มเติม
- ห้ามขยายคำแนะนำไปไม้ประดับชนิดอื่นโดยไม่ตรวจฉลาก

## แหล่งอ้างอิง

ศรีจำนรรจ์ ศรีจันทรา และพฤทธิชาติ ปุญวัฒโท. 2568. _คำแนะนำการใช้สารป้องกันกำจัดศัตรูพืชจากงานวิจัย ปี 2568_. กรมวิชาการเกษตร. หน้า 64.
