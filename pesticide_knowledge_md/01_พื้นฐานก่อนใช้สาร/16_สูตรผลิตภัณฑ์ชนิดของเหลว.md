# สูตรผลิตภัณฑ์ชนิดของเหลว EC SL SC FS CS EW SE OD และ ZC

ผลิตภัณฑ์ที่เห็นเป็นของเหลวไม่ได้มีโครงสร้างเหมือนกัน บางสูตรเป็นสารละลายแท้ บางสูตรเป็นอนุภาคแขวนลอย บางสูตรเป็นหยดน้ำมันในน้ำ หรือเป็นแคปซูลขนาดเล็ก ความแตกต่างนี้มีผลต่อการเขย่า การกวนถัง การตกตะกอน การอุดตัน การระเหย และความเสี่ยงต่อผู้ใช้

## ตารางเปรียบเทียบสูตรชนิดของเหลว

| รหัส | ชื่อโดยย่อ                | ลักษณะเมื่อผสมน้ำ              | ประเด็นใช้งานสำคัญ                                          |
| ---- | ------------------------- | ------------------------------ | ----------------------------------------------------------- |
| EC   | สารผสมน้ำมันข้น           | มักเป็นสีขาวขุ่นคล้ายน้ำนม     | มีตัวทำละลาย ต้องระวังการระเหย การติดไฟ และพิษต่อพืชบางกรณี |
| WSC  | สารผสมข้นละลายน้ำ         | ใส ไม่เกิดน้ำนม                | สารออกฤทธิ์ละลายน้ำได้                                      |
| SL   | สารเหลวละลายน้ำ           | ใสหรือขุ่นเล็กน้อยตามผลิตภัณฑ์ | ผสมง่าย แต่ยังต้องใช้อัตราตามฉลาก                           |
| SC   | สารแขวนลอยเข้มข้น         | ขุ่น มีอนุภาคแขวนลอย           | เขย่าก่อนใช้และกวนถัง ลดการตกตะกอน                          |
| F/FL | สารแขวนลอยข้น             | คล้าย SC                       | เป็นของเหลวเข้มข้นคล้ายครีม                                 |
| FS   | สารแขวนลอยสำหรับคลุกเมล็ด | ใช้ตรงหรือผสมน้ำตามฉลาก        | ห้ามนำไปใช้พ่นทั่วไปหากฉลากไม่อนุญาต                        |
| CS   | แคปซูลแขวนลอย             | แคปซูลเล็กกระจายในน้ำ          | อาจควบคุมการปลดปล่อยสาร ต้องกวนให้สม่ำเสมอ                  |
| EW   | อิมัลชันน้ำมันในน้ำ       | หยดน้ำมันกระจายในน้ำ           | ลักษณะเป็นอิมัลชัน ต้องรักษาความคงตัว                       |
| SE   | สารแขวนลอยผสมอิมัลชัน     | มีทั้งอนุภาคและส่วนอิมัลชัน    | ต้องเขย่าและกวนตามฉลาก                                      |
| OD   | สารแขวนลอยในน้ำมัน        | เป็นของเหลวน้ำมันเข้มข้น       | คุณสมบัติการเกาะและซึมอาจต่างจาก SC                         |
| ZC   | SC ผสม CS                 | มีอนุภาคแขวนลอยและแคปซูล       | ต้องจัดการการกระจายตัวของสององค์ประกอบ                      |

## 1. EC สารผสมน้ำมันข้น

สารออกฤทธิ์ละลายในตัวทำละลายและมีสารช่วยทำอิมัลชัน เมื่อผสมน้ำจะกระจายเป็นหยดเล็กและมักเห็นเป็นสีขาวขุ่น

### จุดเด่น

- ตวงและผสมได้สะดวก
- กระจายตัวในน้ำได้เมื่อสูตรอยู่ในสภาพดี
- สารบางชนิดซึมและเกาะผิวได้ดี

### ข้อควรระวัง

- ตัวทำละลายอาจระคายผิวและเพิ่มการซึมผ่านผิวหนัง
- อาจมีกลิ่นและระเหยง่ายกว่า
- ผลิตภัณฑ์บางชนิดอาจติดไฟ
- อาจเกิดพิษต่อพืชเมื่อใช้เข้มข้น สภาพร้อน หรือผสมหลายชนิด
- ควรตรวจภาชนะ เพราะตัวทำละลายอาจทำให้วัสดุบางชนิดเสื่อม

## 2. SL และ WSC สารละลายน้ำ

สารออกฤทธิ์อยู่ในสภาพละลาย เมื่อเติมน้ำจึงมักได้สารละลายใสและไม่เกิดลักษณะน้ำนมเหมือน EC

### จุดเด่น

- ผสมกับน้ำได้ง่าย
- ไม่มีอนุภาคแข็งแขวนลอย จึงมีความเสี่ยงอุดตันจากตะกอนต่ำกว่าสูตรผง

### ข้อควรระวัง

- คำว่า “ละลายน้ำ” ไม่ได้หมายความว่าปลอดภัยต่อผู้ใช้หรือสิ่งแวดล้อม
- น้ำกระด้าง ค่า pH และการผสมผลิตภัณฑ์อื่นอาจยังมีผล
- ต้องตรวจความเข้ากันได้ก่อนผสมถัง

## 3. SC, F และ FL สารแขวนลอยเข้มข้น

สารออกฤทธิ์ที่ไม่ละลายหรือแทบไม่ละลายในน้ำถูกบดให้มีขนาดเล็กและแขวนลอยในของเหลว สูตรนี้จึงไม่ใช่สารละลายแท้

### วิธีจัดการ

1. เขย่าภาชนะก่อนตวง
2. เติมน้ำบางส่วนลงถัง
3. เติมผลิตภัณฑ์ตามลำดับที่ฉลากกำหนด
4. เปิดระบบกวนหรือกวนถังให้สม่ำเสมอ
5. พ่นให้หมดในเวลาที่เหมาะสม ไม่ปล่อยทิ้งจนตกตะกอน

### สัญญาณปัญหา

- ตะกอนแข็งก้นภาชนะ
- เขย่าแล้วไม่กลับเป็นเนื้อเดียว
- มีก้อนหรือแยกชั้นผิดปกติ
- หัวฉีดอุดตันบ่อย

## 4. FS สูตรสำหรับคลุกเมล็ด

ออกแบบให้เคลือบเมล็ดอย่างสม่ำเสมอ อาจพร้อมใช้หรือผสมน้ำตามฉลาก

ข้อกำหนดสำคัญ

- ใช้เฉพาะเมล็ดและอัตราที่ขึ้นทะเบียน
- เมล็ดคลุกสารห้ามใช้เป็นอาหารหรืออาหารสัตว์
- ติดฉลากและแยกเก็บชัดเจน
- สวมถุงมือและหน้ากากระหว่างคลุก
- ห้ามดัดแปลงไปใช้พ่นทางใบเอง

## 5. CS สูตรแคปซูลแขวนลอย

สารออกฤทธิ์ถูกห่อหุ้มในแคปซูลขนาดเล็กและกระจายในของเหลว การห่อหุ้มอาจช่วยควบคุมการปลดปล่อย ลดกลิ่น หรือเพิ่มระยะออกฤทธิ์ แต่ยังต้องป้องกันการสัมผัสตามฉลาก

## 6. EW, SE, OD และ ZC

เป็นสูตรผสมที่ซับซ้อนกว่าสารละลายทั่วไป

- **EW** มีหยดน้ำมันกระจายในน้ำ
- **SE** รวมคุณสมบัติของสารแขวนลอยกับอิมัลชัน
- **OD** แขวนอนุภาคสารในน้ำมันหรือตัวกลางอินทรีย์
- **ZC** รวมสูตรแคปซูลแขวนลอยกับสารแขวนลอยเข้มข้น

การตัดสินความเข้ากันได้จากการมองว่าเป็น “ยาน้ำเหมือนกัน” จึงไม่เพียงพอ

## 7. การผสมถังแบบลดความเสี่ยง

- อ่านฉลากของทุกผลิตภัณฑ์
- ใช้น้ำสะอาดและตรวจค่า pH หรือความกระด้างเมื่อจำเป็น
- ทดลองผสมปริมาณเล็กในภาชนะใสก่อน
- เติมสารตามลำดับที่ฉลากหรือคำแนะนำรับรอง
- กวนระหว่างเติมและระหว่างพ่น
- หยุดใช้เมื่อเกิดความร้อนผิดปกติ ฟองมาก แยกชั้น ตะกอน หรือก้อน
- ห้ามใช้การทดลองผสมในขวดแทนการตรวจพิษต่อพืช

## หลักที่ควรจำ

> ของเหลวใส ของเหลวขุ่น และของเหลวคล้ายครีมมีโครงสร้างต่างกัน การเขย่าและกวนถังจึงเป็นส่วนหนึ่งของอัตราใช้ที่ถูกต้อง
