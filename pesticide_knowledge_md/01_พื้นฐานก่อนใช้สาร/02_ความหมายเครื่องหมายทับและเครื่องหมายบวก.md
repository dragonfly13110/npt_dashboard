# เครื่องหมายทับและเครื่องหมายบวกในชื่อสารต่างกันอย่างไร

ในเอกสารคำแนะนำจะพบชื่อสารที่มีเครื่องหมาย `/` หรือ `+` คั่นระหว่างชื่อสารออกฤทธิ์ เครื่องหมายทั้งสองแบบไม่ได้ใช้แทนกัน และมีผลต่อการเลือกผลิตภัณฑ์และการผสมสารอย่างมาก

## เครื่องหมาย `/` คือสูตรผสมสำเร็จรูป

เครื่องหมายทับหมายถึง ผลิตภัณฑ์ที่มีสารออกฤทธิ์มากกว่าหนึ่งชนิดและถูกพัฒนาเป็นผลิตภัณฑ์สำเร็จรูปจากผู้ผลิตแล้ว

ตัวอย่างจากหลักการเขียนในเอกสาร

`ไทอะมีทอกแซม/แลมบ์ดา-ไซฮาโลทริน`

สิ่งที่ต้องตรวจสอบ

- ชื่อสารออกฤทธิ์ทั้งสองชนิดตรงตามคำแนะนำ
- สัดส่วนและความเข้มข้นตรงกับสูตรที่ระบุ
- รหัสสูตรผลิตภัณฑ์ตรงกัน
- ผลิตภัณฑ์ขึ้นทะเบียนสำหรับพืชและศัตรูพืชเป้าหมาย

ห้ามซื้อสารเดี่ยวสองขวดมาผสมเองแล้วถือว่าเทียบเท่าผลิตภัณฑ์สูตรผสมสำเร็จรูป เพราะสัดส่วน ตัวทำละลาย สารช่วย และสมบัติทางกายภาพอาจแตกต่างกัน

## เครื่องหมาย `+` คือการผสมในถังพ่น

เครื่องหมายบวกหมายถึง การนำผลิตภัณฑ์ตั้งแต่สองชนิดมาผสมร่วมกันในถังพ่นตามคำแนะนำเฉพาะ

ตัวอย่างจากหลักการเขียนในเอกสาร

`อิมิดาโคลพริด + ไซเพอร์เมทริน`

การผสมในถังต้องพิจารณามากกว่าชื่อสาร เช่น

- ความเข้ากันได้ทางกายภาพและเคมี
- ลำดับการเติมสารลงถัง
- คุณภาพน้ำและค่าความเป็นกรดด่าง
- อัตราของแต่ละผลิตภัณฑ์
- ความเสี่ยงต่อพืชเป็นพิษ
- ความเสี่ยงต่อผู้ผสมและผู้พ่น
- ข้อความบนฉลากของผลิตภัณฑ์แต่ละชนิด

## เปรียบเทียบสองแบบ

| ประเด็น                   | `/` สูตรผสมสำเร็จรูป      | `+` ผสมในถังพ่น    |
| ------------------------- | ------------------------- | ------------------ |
| ผู้กำหนดสัดส่วน           | ผู้ผลิต                   | ผู้ใช้ตามคำแนะนำ   |
| รูปแบบผลิตภัณฑ์           | อยู่ในภาชนะเดียว          | ผลิตภัณฑ์แยกกัน    |
| ความเข้ากันได้            | ผ่านการพัฒนาสูตรผลิตภัณฑ์ | ต้องมีข้อมูลรองรับ |
| ความเสี่ยงจากการตวงผิด    | ต่ำกว่า                   | สูงกว่า            |
| สามารถใช้แทนกันได้หรือไม่ | ไม่ได้                    | ไม่ได้             |

## ข้อควรระวัง

1. อย่าผสมสารเพิ่มเพียงเพราะต้องการให้แรงขึ้น
2. อย่านำสูตรผสมสำเร็จรูปไปผสมสารอื่นต่อโดยไม่มีข้อมูลรองรับ
3. อย่าคิดว่าสารที่ไม่ตกตะกอนแปลว่าเข้ากันได้และปลอดภัยเสมอ
4. การผสมหลายสารอาจเพิ่มความเสี่ยงต่อพืช ผู้ใช้ สารตกค้าง และสิ่งแวดล้อม
5. ต้องตรวจสอบฉลากและคำแนะนำเฉพาะของผลิตภัณฑ์ทุกชนิด

## หลักที่ควรจำ

> `/` คือสารที่ผสมมาแล้วจากโรงงาน ส่วน `+` คือการนำสารมาผสมกันในถังตามคำแนะนำ ทั้งสองแบบไม่สามารถสลับใช้กันเองได้

## แหล่งอ้างอิง

ศรีจำนรรจ์ ศรีจันทรา และพฤทธิชาติ ปุญวัฒโท. 2568. _เอกสารวิชาการ คำแนะนำการใช้สารป้องกันกำจัดศัตรูพืชจากงานวิจัย ปี 2568_. กรมวิชาการเกษตร. คำแนะนำการใช้เอกสาร หน้า ii.
