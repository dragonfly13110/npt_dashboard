# ผลกระทบของสารป้องกันกำจัดศัตรูพืชต่อสุขภาพ สิ่งแวดล้อม และเศรษฐกิจ

การใช้สารป้องกันกำจัดศัตรูพืชช่วยลดการระบาดได้รวดเร็ว แต่การใช้ไม่เหมาะสมก่อให้เกิดต้นทุนที่มองไม่เห็น ทั้งต่อผู้ใช้ ผู้บริโภค ระบบนิเวศ และตลาดสินค้าเกษตร

## 1. ผลกระทบต่อสุขภาพ

ผู้ใช้มีโอกาสได้รับสารขณะเปิดภาชนะ ตวง ผสม พ่น ล้างอุปกรณ์ และจัดการภาชนะใช้แล้ว ส่วนผู้บริโภคอาจได้รับสารจากผลผลิตที่มีสารตกค้าง

ผลกระทบที่อาจพบ ได้แก่

- อาการอ่อนเพลีย วิงเวียน อาเจียน ปวดท้อง หรือท้องร่วงหลังรับสาร
- การรับสารสะสมจากการสัมผัสซ้ำ
- ค่าใช้จ่ายในการรักษาพยาบาล
- การสูญเสียเวลาทำงาน
- อาการแพ้หรือผลกระทบที่ยังคงอยู่หลังการรักษา

แนวทางลดความเสี่ยง

1. ใช้สารเมื่อมีความจำเป็นและตรงกับศัตรูพืช
2. ใช้ตามอัตราและวิธีที่ฉลากกำหนด
3. สวมอุปกรณ์ป้องกันตามลักษณะงาน
4. ตรวจเครื่องพ่นไม่ให้รั่ว
5. รักษาระยะเว้นก่อนเก็บเกี่ยว
6. ล้างผลผลิตก่อนบริโภค

เอกสารระบุว่า การล้างผักด้วยน้ำไหลนาน 2 นาทีลดปริมาณสารพิษได้ประมาณร้อยละ 54-63 อย่างไรก็ตาม ประสิทธิภาพจริงขึ้นกับชนิดสาร ชนิดผลผลิต และวิธีการล้าง การล้างจึงไม่ควรถูกใช้แทนการใช้สารอย่างถูกต้องตั้งแต่ในแปลง

## 2. ผลกระทบต่อสิ่งแวดล้อม

สารอาจเคลื่อนจากพื้นที่เป้าหมายไปสู่ดิน น้ำ อากาศ หรือสิ่งมีชีวิตอื่นผ่านละอองฟุ้ง น้ำไหลบ่า การรั่วไหล และการทิ้งภาชนะไม่ถูกวิธี

ผลกระทบสำคัญ ได้แก่

- สิ่งมีชีวิตในดินและแหล่งน้ำได้รับพิษ
- ปลา นก ผึ้ง และแมลงที่มีประโยชน์ตาย
- ศัตรูธรรมชาติของแมลงศัตรูพืชลดลง
- สารเข้าสู่ห่วงโซ่อาหาร
- สมดุลระบบนิเวศเปลี่ยนแปลง
- ศัตรูพืชบางชนิดกลับมาระบาดมากขึ้น
- ศัตรูพืชชนิดรองอาจกลายเป็นศัตรูสำคัญ

ตัวอย่างเชิงระบบคือ การพ่นสารที่ทำลายทั้งแมลงศัตรูพืชและศัตรูธรรมชาติ เมื่อศัตรูธรรมชาติลดลง แมลงศัตรูพืชที่ฟื้นตัวเร็วอาจกลับมาระบาดรุนแรงกว่าเดิม

## 3. ผลกระทบต่อเศรษฐกิจ

ต้นทุนไม่ได้มีเพียงราคาสารและค่าแรงพ่น แต่รวมถึง

- การพ่นซ้ำเมื่อใช้สารไม่ตรงชนิดศัตรูพืช
- ผลผลิตเสียหายจากการใช้ผิดเวลา
- พืชเกิดอาการเป็นพิษจากสาร
- ค่าใช้จ่ายรักษาพยาบาล
- การสูญเสียรายได้ระหว่างเจ็บป่วย
- ผลผลิตถูกปฏิเสธเมื่อสารตกค้างเกินค่ามาตรฐาน
- การหยุดชะงักของการส่งออก
- การเพิ่มต้นทุนเมื่อศัตรูพืชพัฒนาความต้านทาน

การเพิ่มอัตราหรือพ่นถี่กว่าคำแนะนำจึงอาจเพิ่มต้นทุนทั้งในระยะสั้นและระยะยาว โดยไม่ได้รับประกันว่าจะควบคุมศัตรูพืชได้ดีขึ้น

## 4. ผลกระทบเชื่อมโยงกันอย่างไร

| การปฏิบัติที่ไม่เหมาะสม        | ผลทันที                      | ผลต่อเนื่อง                              |
| ------------------------------ | ---------------------------- | ---------------------------------------- |
| ใช้สารไม่ตรงศัตรูพืช           | ควบคุมไม่ได้                 | ต้องพ่นซ้ำ ต้นทุนสูงขึ้น                 |
| พ่นเกินอัตรา                   | ผู้ใช้และพืชได้รับสารมากขึ้น | เสี่ยงสารตกค้างและพืชเป็นพิษ             |
| พ่นซ้ำกลุ่มเดิม                | ศัตรูพืชที่ไวต่อสารถูกกำจัด  | ตัวที่ทนสารเพิ่มจำนวน เกิดการดื้อสาร     |
| พ่นโดยไม่คำนึงถึงศัตรูธรรมชาติ | แมลงที่มีประโยชน์ลดลง        | ศัตรูพืชกลับมาระบาดหรือเกิดศัตรูชนิดใหม่ |
| ทิ้งน้ำล้างและภาชนะลงแหล่งน้ำ  | น้ำปนเปื้อน                  | กระทบปลา สัตว์น้ำ และห่วงโซ่อาหาร        |
| ไม่รักษาระยะเว้นก่อนเก็บเกี่ยว | ผลผลิตมีโอกาสตกค้างสูง       | กระทบผู้บริโภคและตลาด                    |

## 5. หลักคิดก่อนตัดสินใจพ่น

ควรถามให้ครบก่อนใช้สาร

1. ระบุศัตรูพืชถูกต้องหรือยัง
2. การระบาดถึงระดับที่จำเป็นต้องใช้สารหรือไม่
3. มีวิธีเขตกรรม ชีววิธี หรือวิธีอื่นที่ใช้ร่วมได้หรือไม่
4. สารที่เลือกขึ้นทะเบียนกับพืชและศัตรูเป้าหมายหรือไม่
5. มีทางเลือกที่กระทบศัตรูธรรมชาติน้อยกว่าหรือไม่
6. มีเวลาพอรักษาระยะเว้นก่อนเก็บเกี่ยวหรือไม่
7. ผู้พ่นมีอุปกรณ์ป้องกันและเครื่องพ่นที่พร้อมใช้งานหรือไม่

## หลักที่ควรจำ

> การใช้สารที่คุ้มค่า ต้องควบคุมศัตรูพืชได้โดยไม่ผลักต้นทุนไปให้สุขภาพ สิ่งแวดล้อม และตลาดสินค้าเกษตร
