# อุปกรณ์ป้องกันส่วนบุคคลสำหรับการใช้สารป้องกันกำจัดศัตรูพืช

อุปกรณ์ป้องกันส่วนบุคคลช่วยลดการสัมผัสสารทางผิวหนัง ดวงตา และระบบหายใจ ต้องเลือกให้เหมาะกับงาน ชนิดผลิตภัณฑ์ และข้อความบนฉลาก อุปกรณ์ที่สวมไม่ครบ ชำรุด หรือใช้ผิดชนิดอาจให้ความมั่นใจที่ผิดและยังรับสารเข้าสู่ร่างกายได้

## 1. ชุดป้องกันร่างกาย

ชุดที่เหมาะสมควรปกคลุมร่างกายมากที่สุด มีความคงทน และทำความสะอาดได้ง่าย ชุดคลุมทั้งตัวเหมาะกับงานที่มีโอกาสสัมผัสละอองหรือสารเข้มข้นสูง

กรณีไม่มีชุดเฉพาะ อย่างน้อยควรใช้

- เสื้อแขนยาว
- กางเกงขายาว
- เสื้อผ้าที่อยู่ในสภาพดี ไม่มีรอยขาด
- หมวกที่ป้องกันศีรษะและลำคอ

เสื้อผ้าไม่ควรหนาจนทำให้เกิดความเครียดจากความร้อน และควรซักแยกจากเสื้อผ้าของสมาชิกในครอบครัว

## 2. ถุงมือ

ถุงมือต้องทนต่อสารและตัวทำละลายในผลิตภัณฑ์ ถุงมือผ้าดูดซับสารและไม่เหมาะกับการตวงหรือผสมสารเข้มข้น

ก่อนใช้ทุกครั้ง

1. ตรวจรอยแตก รั่ว และชำรุด โดยเฉพาะซอกนิ้ว
2. เลือกขนาดที่กระชับและหยิบจับอุปกรณ์ได้
3. ใส่แขนเสื้อให้สัมพันธ์กับลักษณะงาน เพื่อไม่ให้สารไหลเข้าสู่ถุงมือ
4. หลังใช้ ล้างภายนอกก่อนถอด ลดโอกาสสัมผัสผิวหนัง
5. ทำความสะอาดทั้งด้านนอกและด้านใน แล้วผึ่งให้แห้ง

## 3. รองเท้าบู๊ต

สำหรับงานในแปลงหรือในนาข้าว ควรใช้รองเท้าบู๊ตที่กระชับ ไม่มีซับใน และสูงเพียงพอป้องกันละอองและน้ำในแปลง

ควรสวมขากางเกงคลุมด้านนอกรองเท้า เพื่อป้องกันสารไหลลงไปขังภายในรองเท้า ตรวจรอยรั่วและล้างทุกครั้งหลังใช้งาน

## 4. อุปกรณ์ปกป้องระบบหายใจ

เอกสารแบ่งเป็น 2 แบบหลัก

### หน้ากากชนิดใช้แล้วทิ้ง

หน้ากากที่ใช้กับงานพ่นสารควรมีส่วนกรองละอองและส่วนกรองไอระเหยตามความเหมาะสม หน้ากากกันฝุ่นทั่วไปอาจไม่สามารถกรองไอระเหยของสารอินทรีย์ได้

### หน้ากากชนิดเปลี่ยนไส้กรอง

ประกอบด้วยตัวหน้ากาก แผ่นกรองอนุภาค และตลับกรองที่เหมาะกับไอระเหย หน้ากากต้องแนบสนิทกับใบหน้าและเปลี่ยนไส้กรองตามคำแนะนำของผู้ผลิต

ข้อควรระวัง

- หนวด เครา หรือรูปหน้าที่ทำให้หน้ากากไม่แนบสนิทลดประสิทธิภาพการป้องกัน
- ตลับกรองหมดอายุได้แม้ยังดูสะอาด
- หากได้กลิ่นสาร หายใจลำบาก หรือหน้ากากชำรุด ต้องออกจากพื้นที่และตรวจสอบทันที

## 5. ครอบตานิรภัย

ควรสวมขณะตวง ผสม เทถ่าย และพ่นสาร เพื่อป้องกันละอองกระเด็นเข้าตาและผิวหนังรอบดวงตา

ลักษณะที่ควรเลือก

- กระชับกับรูปหน้า
- แข็งแรงและใช้งานได้ต่อเนื่อง
- ผ่านมาตรฐานที่เชื่อถือได้
- ทำความสะอาดง่าย
- ทนต่อความร้อนและไม่ติดไฟง่าย

แว่นสายตาทั่วไปไม่ทดแทนครอบตานิรภัย เพราะมีช่องเปิดด้านข้างและไม่ป้องกันละอองรอบดวงตา

## 6. ผ้ากันเปื้อน

ใช้ป้องกันด้านหน้าของร่างกายขณะตวง ผสม เทถ่าย หรือล้างอุปกรณ์ ควรทำจากวัสดุที่ไม่ดูดซับสาร เช่น พลาสติก ยาง หรือวัสดุสังเคราะห์ที่เหมาะสม และคลุมตั้งแต่หน้าอกถึงหัวเข่า

## ชุดขั้นต่ำตามลักษณะงาน

| งาน                   | อุปกรณ์ขั้นต่ำที่ควรมี                                                        |
| --------------------- | ----------------------------------------------------------------------------- |
| ขนย้ายภาชนะที่ปิดสนิท | เสื้อแขนยาว กางเกงขายาว รองเท้าหุ้มเท้า ถุงมือเมื่อมีความเสี่ยงรั่ว           |
| เปิด ตวง และผสมสาร    | ชุดปกคลุมร่างกาย ถุงมือทนสาร รองเท้าบู๊ต ครอบตา หน้ากากตามฉลาก ผ้ากันเปื้อน   |
| พ่นสาร                | เสื้อแขนยาว กางเกงขายาว ถุงมือ รองเท้าบู๊ต หมวก ครอบตา และอุปกรณ์หายใจตามฉลาก |
| ล้างอุปกรณ์           | ถุงมือ รองเท้าบู๊ต ครอบตา ผ้ากันเปื้อน และชุดที่ป้องกันน้ำกระเด็น             |

## หลังเลิกงาน

1. ล้างอุปกรณ์ป้องกันภายนอกก่อนถอด
2. ถอดโดยหลีกเลี่ยงการจับด้านที่ปนเปื้อน
3. ล้างมือและอาบน้ำทันที
4. ซักเสื้อผ้าทำงานแยกจากเสื้อผ้าทั่วไป
5. ผึ่งอุปกรณ์ให้แห้งในพื้นที่ที่ไม่ปนเปื้อนอาหารและของใช้
6. ตรวจสภาพก่อนเก็บ หากชำรุดให้เปลี่ยน

## หลักที่ควรจำ

> อุปกรณ์ป้องกันส่วนบุคคลเป็นแนวป้องกันสุดท้าย ไม่สามารถชดเชยการใช้สารผิดอัตรา อุปกรณ์พ่นรั่ว หรือการพ่นทวนลมได้
