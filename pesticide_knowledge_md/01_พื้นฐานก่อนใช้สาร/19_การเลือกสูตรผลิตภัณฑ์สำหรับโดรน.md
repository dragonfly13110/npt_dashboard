# การเลือกสูตรผลิตภัณฑ์สารป้องกันกำจัดศัตรูพืชสำหรับโดรน

โดรนใช้ปริมาณน้ำต่ำ ระบบปั๊ม ท่อ กรอง และหัวฉีดมีขนาดเล็กกว่าการพ่นภาคพื้นดินหลายแบบ ความเข้มข้นในถังจึงสูงขึ้นและความผิดพลาดเรื่องการตกตะกอน ความหนืด ฟอง และการอุดตันเห็นผลเร็ว การเลือกสูตรต้องพิจารณาทั้งประสิทธิภาพต่อศัตรูพืชและความเข้ากันได้กับระบบพ่น

## 1. หลักการเลือกเบื้องต้นตามเอกสาร

ควรเลือกผลิตภัณฑ์ที่เมื่อผสมน้ำแล้วกระจายเป็นเนื้อเดียวหรือคงตัวได้ดี และเป็นผลิตภัณฑ์ที่ฉลากหรือคำแนะนำอนุญาตให้ใช้ในลักษณะดังกล่าว

เอกสารให้ข้อสังเกตว่า

- ไม่ควรนำสูตร G หรือ GR ซึ่งออกแบบให้หว่านลงดินมาแช่น้ำเพื่อพ่น
- สูตร WP และ WG ต้องระวังการตกตะกอนและการอุดตันของปั๊มหรือหัวฉีด
- สูตร SC และ SL มีแนวโน้มลดการปลิวและการระเหยได้ดีกว่าสูตร EC ในการเปรียบเทียบตามที่เอกสารกล่าวถึง

ข้อสังเกตนี้เป็นหลักพิจารณา ไม่ได้หมายความว่า SC หรือ SL ทุกผลิตภัณฑ์ใช้กับโดรนได้โดยอัตโนมัติ

## 2. เหตุผลที่ห้ามนำ GR มาแช่น้ำพ่น

สูตร GR ออกแบบให้สารออกฤทธิ์เคลือบหรือผสมกับเม็ดพาหะสำหรับหว่าน โรย หรือหยอด

การนำไปแช่น้ำอาจทำให้

- สารออกฤทธิ์ออกมาไม่ครบหรือไม่สม่ำเสมอ
- เศษดิน ทราย หรือวัสดุพาหะเข้าสู่ระบบ
- กรอง ปั๊ม และหัวฉีดอุดตันหรือสึกหรอ
- ไม่สามารถคำนวณอัตราสารต่อพื้นที่ได้ถูกต้อง
- เป็นการใช้ผิดวิธีจากฉลาก

## 3. การประเมิน WP และ WG

WP และ WG สามารถใช้ผสมน้ำได้ตามสูตร แต่ไม่ได้ละลายเป็นสารละลายแท้ อนุภาคจึงอาจตกตะกอนในถังหรือสายเมื่อระบบกวนไม่เพียงพอ

ก่อนใช้กับโดรนต้องตรวจอย่างน้อย

- ขนาดอนุภาคหลังผสม
- เวลาที่ใช้กระจายตัว
- ความคงตัวหลังพัก
- ระบบกวนในถัง
- ขนาดตะแกรงกรอง
- ขนาดช่องหัวฉีด
- การอุดตันในการทดสอบด้วยน้ำยา
- คำแนะนำจากฉลาก ผู้ผลิต และผู้ให้บริการโดรน

## 4. การประเมิน SC และ SL

### SC

- เป็นอนุภาคแขวนลอยเข้มข้น
- ต้องเขย่าภาชนะและกวนถัง
- อาจตกตะกอนเมื่อหยุดพักนาน
- ต้องตรวจความเข้ากันได้กับสารอื่นและสารช่วย

### SL

- สารอยู่ในสภาพละลาย จึงลดความเสี่ยงตะกอนจากอนุภาค
- ยังอาจเกิดผลึกเมื่อความเข้มข้นสูง น้ำไม่เหมาะสม หรือผสมสารอื่น
- ความหนืด ฟอง และการกัดกร่อนยังต้องตรวจ

## 5. EC กับประเด็นการปลิวและการระเหย

EC มีตัวทำละลายอินทรีย์และเมื่อผสมน้ำเกิดอิมัลชัน เอกสารระบุว่า SC และ SL มีคุณสมบัติด้านการลดการปลิวและการระเหยดีกว่า EC อย่างไรก็ตาม การปลิวจริงขึ้นกับปัจจัยอื่นด้วย เช่น

- ขนาดละออง
- ชนิดหัวฉีด
- ความสูงบิน
- ความเร็วบิน
- แรงลม
- อุณหภูมิและความชื้น
- สารช่วยลดการปลิว
- อัตราการพ่น

การเปลี่ยนสูตรอย่างเดียวจึงไม่แก้ปัญหาการปลิวทั้งหมด

## 6. ขั้นตอนทดสอบก่อนนำขึ้นบิน

1. ตรวจฉลากว่าใช้กับพืช ศัตรู และวิธีพ่นที่ต้องการได้
2. คำนวณปริมาณสารต่อไร่ ไม่คำนวณจากความจุถังอย่างเดียว
3. ใช้น้ำที่จะใช้จริงในการทดสอบ
4. ทดลองผสมในภาชนะใสตามลำดับการผสม
5. สังเกตตะกอน ก้อน ฟอง ความร้อน และการแยกชั้น
6. กรองและหมุนเวียนผ่านระบบภาคพื้นดิน
7. ตรวจการอุดตัน อัตราการไหล และรูปแบบละออง
8. สอบเทียบความเร็ว ความสูง แนวบิน และระยะห่างแนวพ่น
9. ทดสอบพื้นที่เล็กและประเมินการคลุมเป้าหมาย
10. ล้างระบบทันทีหลังใช้งาน

## 7. แบบตรวจสอบก่อนบินพ่น

- [ ] ผู้ควบคุมและการบินเป็นไปตามกฎหมาย
- [ ] ผลิตภัณฑ์ขึ้นทะเบียนและวิธีใช้รองรับ
- [ ] สูตรไม่ใช่ GR ที่นำมาดัดแปลงแช่น้ำ
- [ ] ทดสอบความเข้ากันได้แล้ว
- [ ] ไม่มีตะกอน ก้อน หรือฟองผิดปกติ
- [ ] กรอง ปั๊ม และหัวฉีดรองรับสูตร
- [ ] สอบเทียบลิตรต่อไร่แล้ว
- [ ] คำนวณปริมาณสารต่อพื้นที่แล้ว
- [ ] สภาพลม อุณหภูมิ และความชื้นอยู่ในเกณฑ์
- [ ] มีเขตกันชนและป้องกันคน สัตว์ แหล่งน้ำ และแปลงข้างเคียง
- [ ] มีแผนจัดการน้ำยาคงเหลือและน้ำล้าง

## 8. จุดที่มักทำผิด

| ความผิดพลาด                       | ผลที่อาจเกิด                           |
| --------------------------------- | -------------------------------------- |
| คิดว่าสูตรน้ำทุกชนิดใช้กับโดรนได้ | อุดตัน แยกชั้น หรือพืชเป็นพิษ          |
| ลดน้ำโดยคงอัตราต่อถังเดิม         | สารต่อไร่สูงเกิน                       |
| ไม่กวนถัง SC, WP หรือ WG          | ความเข้มข้นต้นถังและท้ายถังต่างกัน     |
| ผสมหลายชนิดโดยไม่ทดสอบ            | ตะกอน ฟอง ความหนืด หรือประสิทธิภาพลดลง |
| บินสูงเพื่อทำงานเร็ว              | ละอองปลิวและคลุมเป้าหมายลดลง           |
| ใช้หัวฉีดเล็กเพื่อให้ละอองฝอยมาก  | การปลิวและการอุดตันเพิ่มขึ้น           |

## หลักที่ควรจำ

> สูตรที่เหมาะกับโดรนต้องผ่านทั้งสามเงื่อนไข คือใช้ได้ตามฉลาก คงตัวในน้ำเข้มข้น และผ่านระบบปั๊ม กรอง และหัวฉีดโดยไม่เปลี่ยนอัตราสารต่อพื้นที่
