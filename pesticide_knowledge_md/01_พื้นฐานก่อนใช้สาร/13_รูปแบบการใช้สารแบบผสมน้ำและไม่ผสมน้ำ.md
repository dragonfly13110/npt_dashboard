# รูปแบบการใช้สารป้องกันกำจัดศัตรูพืชแบบผสมน้ำและไม่ผสมน้ำ

การแบ่งวิธีพ่นตามปริมาณน้ำไม่ได้บอกว่าวิธีใดดีกว่าเสมอไป แต่ใช้ช่วยเลือกเครื่องพ่น หัวฉีด ขนาดละออง ความเร็วเดิน และความเข้มข้นของสารให้สัมพันธ์กัน เป้าหมายสำคัญคือให้ปริมาณสารออกฤทธิ์ต่อพื้นที่ถูกต้องและละอองไปถึงตำแหน่งที่ศัตรูพืชอาศัยอยู่

## 1. หลักที่ต้องแยกให้ออก

- **อัตราสาร** คือปริมาณผลิตภัณฑ์หรือสารออกฤทธิ์ที่ใช้ต่อพื้นที่หรือต่อน้ำตามฉลาก
- **อัตราการพ่น** คือปริมาณน้ำผสมสารที่เครื่องพ่นปล่อยต่อพื้นที่ เช่น ลิตรต่อไร่
- **ความเข้มข้นในถัง** จะเปลี่ยนเมื่อปริมาณน้ำเปลี่ยน แต่ปริมาณสารต่อไร่ต้องยังเป็นไปตามคำแนะนำ
- การลดน้ำโดยไม่สอบเทียบเครื่องและไม่คำนวณใหม่ อาจทำให้ได้รับสารต่อพื้นที่มากหรือน้อยเกินไป

## 2. การใช้แบบผสมน้ำมาก

ใช้ปริมาณน้ำมาก ละอองมักมีขนาดค่อนข้างใหญ่ เหมาะกับเครื่องพ่นชนิดใช้แรงคนหรือเครื่องยนต์ที่ต้องการให้น้ำยาครอบคลุมทรงพุ่ม

### จุดเด่น

- มองเห็นการเปียกของพืชได้ง่าย
- เหมาะกับผู้ใช้ที่คุ้นเคยกับเครื่องพ่นทั่วไป
- ช่วยให้ละอองเข้าถึงทรงพุ่มเมื่อปรับหัวฉีดและแรงดันเหมาะสม

### ข้อจำกัด

- ใช้น้ำและเวลาเติมถังมาก
- ถ้าพ่นจนโชก น้ำยาจะรวมตัวไหลลงดิน
- สารที่ไหลออกจากใบไม่ได้ช่วยเพิ่มประสิทธิภาพ แต่เพิ่มการสูญเสียและการปนเปื้อน

> พ่นให้ทั่วถึง แต่ไม่ควรพ่นจนมีน้ำยาไหลหยดจากใบและทรงพุ่ม

## 3. การใช้แบบผสมน้ำปานกลาง

เป็นช่วงที่พบได้บ่อยกับเครื่องพ่นสะพายหลังทั้งแบบแรงคนและเครื่องยนต์ ลดปริมาณน้ำจากแบบผสมน้ำมาก แต่ยังคงให้การคลุมเป้าหมายที่ดีเมื่อสอบเทียบเครื่องถูกต้อง

สิ่งที่ต้องควบคุม ได้แก่

- อัตราการไหลของหัวฉีด
- ความเร็วเดิน
- ระยะห่างหัวฉีดจากเป้าหมาย
- แรงดันหรือรอบเครื่อง
- จำนวนหัวฉีดและแนวการพ่น

## 4. การใช้แบบผสมน้ำน้อย

ใช้ปริมาณน้ำลดลงและสร้างละอองขนาดเล็กลง มักใช้เครื่องยนต์พ่นสารสะพายหลังแบบใช้แรงลมหรือระบบที่ควบคุมอัตราการไหลได้

### จุดเด่น

- ทำงานได้เร็วขึ้น
- ลดเวลาขนน้ำและเติมถัง
- อาจลดต้นทุนแรงงาน
- ละอองมีความสม่ำเสมอเมื่อเครื่องและหัวฉีดเหมาะสม

### ความเสี่ยง

- ละอองเล็กปลิวออกนอกเป้าหมายได้ง่าย
- ผู้พ่นและคนใกล้เคียงมีโอกาสสัมผัสละอองมากขึ้น
- ความผิดพลาดในการคำนวณหรือสอบเทียบส่งผลรุนแรงกว่าแบบใช้น้ำมาก
- อากาศร้อนและแห้งทำให้ละอองระเหยก่อนถึงเป้าหมาย

## 5. การใช้แบบผสมน้ำน้อยมาก

ใช้น้ำในปริมาณต่ำมาก ต้องอาศัยเครื่องพ่นและหัวฉีดที่ออกแบบมาสำหรับงานนี้ การพ่นให้สำเร็จขึ้นกับการควบคุมขนาดละออง อัตราการไหล ความเร็ว และสภาพอากาศอย่างเข้มงวด

ไม่ควรดัดแปลงจากวิธีพ่นเดิมด้วยการลดน้ำเพียงอย่างเดียว เพราะอาจทำให้

- ความเข้มข้นในถังสูงเกิน
- การกระจายตัวไม่สม่ำเสมอ
- เกิดพิษต่อพืช
- สารตกค้างสูง
- ละอองปลิวไกล

## 6. การใช้แบบไม่ผสมน้ำ

ใช้ผลิตภัณฑ์สูตรเฉพาะสำหรับการพ่นโดยตรง เช่น สูตรที่กำหนดให้ใช้ในระบบปริมาตรต่ำมาก ต้องใช้เครื่องพ่นและหัวฉีดเฉพาะ ไม่ควรนำผลิตภัณฑ์ทั่วไปมาตัดน้ำเองแล้วเรียกว่าเป็นการพ่นแบบไม่ผสมน้ำ

## 7. ตารางช่วงอัตราการพ่นตามเอกสาร

| วิธีการพ่น    | คำย่อ |    พืชไร่ ข้าว และผัก |            ไม้ผลและไม้ยืนต้น |
| ------------- | ----: | --------------------: | ---------------------------: |
| ผสมน้ำมาก     |    HV |   มากกว่า 96 ลิตร/ไร่ |         มากกว่า 160 ลิตร/ไร่ |
| ผสมน้ำปานกลาง |    MV |        32-96 ลิตร/ไร่ |              80-160 ลิตร/ไร่ |
| ผสมน้ำน้อย    |    LV |         8-32 ลิตร/ไร่ |               32-80 ลิตร/ไร่ |
| ผสมน้ำน้อยมาก |   VLV |        0.8-8 ลิตร/ไร่ |                8-32 ลิตร/ไร่ |
| ไม่ผสมน้ำ     |   ULV | น้อยกว่า 0.8 ลิตร/ไร่ | เอกสารระบุมากกว่า 8 ลิตร/ไร่ |

## 8. ข้อสังเกตเรื่องตัวเลขในเอกสาร

ตัวเลขในคำอธิบายเนื้อหาและตัวเลขในตารางมีความต่างกันเล็กน้อย เช่น แบบผสมน้ำมากในเนื้อหาใช้เกณฑ์มากกว่า 80 ลิตรต่อไร่สำหรับพืชไร่ แต่ตารางอ้างอิงกำหนดมากกว่า 96 ลิตรต่อไร่ จึงควรใช้ตัวเลขเหล่านี้เพื่อเข้าใจการจำแนกวิธีพ่น ไม่ควรใช้แทนอัตราบนฉลากหรือผลการสอบเทียบเครื่อง

## 9. วิธีเลือกปริมาณน้ำให้เหมาะกับแปลง

พิจารณาร่วมกันอย่างน้อย 7 เรื่อง

1. ชนิดพืชและความหนาแน่นของทรงพุ่ม
2. ตำแหน่งของศัตรูพืช เช่น ใต้ใบ ยอด ผล หรือโคนต้น
3. คุณสมบัติของสารและสูตรผลิตภัณฑ์
4. เครื่องพ่นและหัวฉีดที่มี
5. ความเร็วเดินหรือความเร็วบิน
6. สภาพลม อุณหภูมิ และความชื้น
7. ความสามารถในการสอบเทียบและควบคุมคุณภาพงานพ่น

## 10. ขั้นตอนสอบเทียบแบบพื้นฐาน

1. เติมน้ำสะอาดและตรวจหารอยรั่ว
2. ใช้หัวฉีด แรงดัน และความเร็วเดียวกับที่จะพ่นจริง
3. พ่นพื้นที่ทดลองที่ทราบขนาด
4. วัดปริมาณน้ำที่ใช้
5. คำนวณเป็นลิตรต่อไร่
6. ปรับหัวฉีด แรงดัน หรือความเร็ว แล้วทดสอบซ้ำ
7. เมื่อได้อัตราการพ่นแล้ว จึงคำนวณสารตามอัตราต่อพื้นที่บนฉลาก

## จุดที่มักทำผิด

| ความผิดพลาด                                     | ผลที่อาจเกิด                 |
| ----------------------------------------------- | ---------------------------- |
| ลดน้ำ แต่ใส่สารเท่าเดิมต่อถังโดยไม่คำนวณพื้นที่ | สารต่อไร่อาจสูงเกิน          |
| ใช้หัวฉีดสึกหรือหัวฉีดต่างขนาดปะปนกัน           | การกระจายไม่สม่ำเสมอ         |
| พ่นจนโชกเพื่อให้มั่นใจ                          | น้ำยาไหลลงดินและสิ้นเปลือง   |
| เพิ่มแรงดันมากเพื่อให้ละอองละเอียด              | ละอองปลิวและหัวฉีดสึกเร็ว    |
| พ่นในลมแรงหรืออากาศร้อนจัด                      | ละอองออกนอกเป้าหมายหรือระเหย |

## หลักที่ควรจำ

> ปริมาณน้ำน้อยลงไม่ได้แปลว่าใช้สารน้อยลง และปริมาณน้ำมากขึ้นไม่ได้แปลว่าควบคุมได้ดีกว่า งานพ่นที่ดีต้องคุมปริมาณสารต่อพื้นที่และการคลุมเป้าหมายให้ถูกต้อง
