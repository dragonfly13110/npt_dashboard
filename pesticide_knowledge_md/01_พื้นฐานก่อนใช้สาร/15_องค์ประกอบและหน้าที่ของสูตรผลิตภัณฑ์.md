# องค์ประกอบและหน้าที่ของสูตรผลิตภัณฑ์สารป้องกันกำจัดศัตรูพืช

สารออกฤทธิ์เพียงอย่างเดียวมักไม่เหมาะสำหรับนำไปใช้โดยตรง เพราะอาจไม่ละลายน้ำ มีความเข้มข้นสูง กระจายตัวไม่สม่ำเสมอ ระเหยง่าย หรือเกาะผิวเป้าหมายไม่ดี จึงต้องผลิตเป็นสูตรที่เหมาะกับการขนส่ง การเก็บรักษา การผสม และการใช้

## 1. สูตรผลิตภัณฑ์คืออะไร

สูตรผลิตภัณฑ์คือรูปแบบของผลิตภัณฑ์ที่เกิดจากการผสมสารออกฤทธิ์กับส่วนประกอบอื่น เพื่อให้สามารถนำไปใช้ควบคุมศัตรูพืชได้อย่างมีประสิทธิภาพและปลอดภัยขึ้น

ตัวอย่างรหัสสูตรที่พบได้บ่อย ได้แก่

- EC สารผสมน้ำมันข้น
- SL สารเหลวละลายน้ำ
- SC สารแขวนลอยเข้มข้น
- WP ผงผสมน้ำ
- WG เม็ดกระจายน้ำ
- GR เม็ดใช้หว่านหรือหยอด
- CS แคปซูลแขวนลอย

รหัสสูตรไม่ได้บอกกลไกการออกฤทธิ์และไม่ได้บอกว่าผลิตภัณฑ์ปลอดภัยกว่าสูตรอื่นเสมอไป

## 2. ส่วนประกอบหลักของผลิตภัณฑ์

### 2.1 สารออกฤทธิ์

เป็นส่วนที่ออกฤทธิ์ต่อโรค แมลง ไร สัตว์ศัตรูพืช หรือวัชพืช บนฉลากมักระบุเป็นชื่อสามัญและเปอร์เซ็นต์ เช่น 10% หรือ 50%

สารออกฤทธิ์แต่ละชนิดมีคุณสมบัติต่างกัน เช่น

- ของแข็งหรือของเหลว
- ละลายน้ำได้มากหรือน้อย
- ละลายในน้ำมันหรือตัวทำละลายอื่น
- ดูดซึมเข้าสู่พืชหรือออกฤทธิ์สัมผัส
- คงตัวหรือสลายตัวง่าย

### 2.2 ส่วนผสมอื่น

เป็นส่วนที่ช่วยให้ผลิตภัณฑ์อยู่ในรูปที่ใช้ได้จริง เช่น

- ตัวทำละลาย
- สารทำให้เจือจาง
- สารช่วยให้เปียก
- สารช่วยกระจายตัว
- สารช่วยแขวนลอย
- สารทำอิมัลชัน
- สารลดแรงตึงผิว
- วัสดุพาหะ
- สารดึงดูดหรืออาหารในสูตรเหยื่อพิษ

คำว่า “ส่วนผสมอื่น” ไม่ได้หมายความว่าไม่มีผลต่อความปลอดภัย เพราะตัวทำละลายหรือสารปรุงแต่งบางชนิดอาจทำให้ระคายเคือง เพิ่มการซึมผ่านผิวหนัง หรือทำให้ผลิตภัณฑ์ติดไฟได้

## 3. เหตุผลที่ต้องปรุงเป็นสูตร

เอกสารอธิบายเป้าหมายหลัก 2 ประการ

1. ทำให้สารออกฤทธิ์กระจายครอบคลุมพื้นที่เป้าหมายอย่างทั่วถึงและสม่ำเสมอ
2. เสริมประสิทธิภาพและคุณสมบัติการใช้งาน

ผลที่ต้องการจากการปรุงสูตรอาจรวมถึง

- ช่วยให้สารที่ไม่ละลายน้ำรวมกับน้ำได้
- ทำให้ละอองแผ่บนผิวใบ
- เพิ่มการจับเกาะเป้าหมาย
- ลดการตกตะกอน
- เพิ่มความคงตัวระหว่างเก็บรักษา
- ลดการปลิวหรือการระเหย
- ควบคุมการปลดปล่อยสารออกฤทธิ์
- ลดการสัมผัสสารเข้มข้นของผู้ใช้

## 4. คุณสมบัติของส่วนผสมปรุงแต่งที่ดี

ตามหลักที่เอกสารระบุ ส่วนผสมควร

- มีต้นทุนเหมาะสม
- ใช้งานง่าย
- สะดวกต่อการเก็บและขนส่ง
- คงสภาพได้นานพอสมควร
- ช่วยให้สารรวมกับน้ำหรือกระจายในน้ำได้
- ช่วยให้สารติดผิวศัตรูพืชหรือพืชเป้าหมาย
- ลดแรงตึงผิวเพื่อให้แผ่กระจายบนใบ

## 5. สูตรเดียวกันไม่ได้แปลว่าใช้แทนกันได้

ตัวอย่าง ผลิตภัณฑ์สองยี่ห้ออาจเป็นสูตร SC เหมือนกัน แต่ต่างกันด้าน

- สารออกฤทธิ์
- ความเข้มข้น
- สารช่วยและความคงตัว
- พืชและศัตรูที่ขึ้นทะเบียน
- อัตราใช้
- ระยะเว้นก่อนเก็บเกี่ยว
- ข้อจำกัดการผสม

จึงห้ามใช้อัตราของผลิตภัณฑ์หนึ่งกับอีกผลิตภัณฑ์หนึ่งเพียงเพราะรหัสสูตรเหมือนกัน

## 6. วิธีอ่านชื่อผลิตภัณฑ์แบบครบองค์ประกอบ

ตัวอย่างรูปแบบการอ่าน

`ชื่อสามัญ + ความเข้มข้น + รหัสสูตร`

เช่น `สาร ก 10% SC`

แปลว่า

- สาร ก คือชื่อสารออกฤทธิ์
- 10% คือสัดส่วนสารออกฤทธิ์ตามรูปแบบที่ฉลากกำหนด
- SC คือสูตรแขวนลอยเข้มข้น ต้องเขย่าและผสมน้ำตามฉลาก

ทั้งสามส่วนต้องตรงกันก่อนนำอัตราใช้มาเปรียบเทียบ

## 7. สูตรผลิตภัณฑ์มีผลต่อการทำงานพ่นอย่างไร

| คุณสมบัติ               | ผลต่อการใช้งาน                                  |
| ----------------------- | ----------------------------------------------- |
| การละลายหรือกระจายในน้ำ | มีผลต่อความเป็นเนื้อเดียวและการอุดตัน           |
| ความหนืด                | มีผลต่ออัตราการไหลและการผสม                     |
| ขนาดอนุภาค              | มีผลต่อการแขวนลอย การตกตะกอน และหัวฉีด          |
| ตัวทำละลาย              | มีผลต่อการระเหย กลิ่น การติดไฟ และการซึมผ่านผิว |
| สารลดแรงตึงผิว          | มีผลต่อการแผ่และการเกาะใบ                       |
| การห่อหุ้มแคปซูล        | มีผลต่อการปลดปล่อยและระยะออกฤทธิ์               |

## 8. คำถามก่อนเลือกสูตร

1. ฉลากระบุให้ใช้กับพืชและศัตรูชนิดนี้หรือไม่
2. สูตรต้องผสมน้ำ ใช้โดยตรง คลุกเมล็ด หรือหว่านลงดิน
3. เครื่องพ่นและหัวฉีดรองรับอนุภาคหรือความหนืดหรือไม่
4. สูตรมีโอกาสตกตะกอนหรือแยกชั้นหรือไม่
5. ต้องกวนถังต่อเนื่องหรือเขย่าก่อนใช้หรือไม่
6. มีข้อกำหนดเรื่องน้ำ คุณภาพน้ำ หรือสารช่วยหรือไม่
7. มีความเสี่ยงต่อผู้ใช้ พืช และพื้นที่ข้างเคียงอย่างไร

## หลักที่ควรจำ

> สารออกฤทธิ์บอกว่าอะไรเป็นตัวควบคุมศัตรูพืช ส่วนรหัสสูตรบอกว่าสารนั้นถูกจัดทำให้อยู่ในรูปใดและต้องจัดการอย่างไร
