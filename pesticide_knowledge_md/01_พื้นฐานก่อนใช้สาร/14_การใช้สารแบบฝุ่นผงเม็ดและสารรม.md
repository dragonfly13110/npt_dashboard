# การใช้สารแบบฝุ่น ผง เม็ด และสารรม

สารป้องกันกำจัดศัตรูพืชไม่ได้มีเฉพาะชนิดผสมน้ำพ่นบนใบ รูปแบบฝุ่น ผง เม็ด และสารรมมีวิธีใช้ ความเสี่ยง และอุปกรณ์แตกต่างกัน การใช้ผิดรูปแบบอาจทำให้ไม่ได้ผล เกิดพิษต่อผู้ใช้ หรือปนเปื้อนสิ่งแวดล้อม

## 1. การพ่นแบบฝุ่นหรือผงโดยไม่ผสมน้ำ

ผลิตภัณฑ์ชนิดฝุ่นหรือผงบางสูตรออกแบบให้ใช้โดยตรง สามารถใช้เครื่องพ่นฝุ่นผงเฉพาะ หรือเครื่องยนต์สะพายหลังชนิดใช้แรงลมที่มีชุดอุปกรณ์สำหรับฝุ่นผง

### สภาพที่เหมาะตามเอกสาร

- พื้นที่ขนาดเล็ก
- พื้นที่หาน้ำยาก
- ลมสงบ
- พืชมีความชื้นเล็กน้อย ช่วยให้ผงเกาะผิวพืช
- ช่วงเช้ามืดหรือกลางคืนซึ่งลมมักสงบกว่า

### ข้อจำกัดสำคัญ

- ประสิทธิภาพอาจต่ำกว่าการใช้แบบผสมน้ำ
- ฝุ่นฟุ้งและปลิวออกนอกเป้าหมายง่าย
- เสี่ยงต่อระบบหายใจของผู้พ่นและผู้ที่อยู่ใกล้เคียง
- การกระจายตัวไม่สม่ำเสมอเมื่อมีลม

### การป้องกัน

- ใช้หน้ากากกรองอนุภาคที่เหมาะสมกับผลิตภัณฑ์
- สวมแว่นหรือครอบตา ถุงมือ ชุดแขนยาว และรองเท้าปิด
- ห้ามพ่นทวนลม
- กันคน สัตว์ และอาหารออกจากพื้นที่
- หยุดพ่นเมื่อฝุ่นปลิวออกนอกเขตเป้าหมาย

## 2. การใช้สารชนิดเม็ด

สารชนิดเม็ดได้จากการนำสารออกฤทธิ์เคลือบบนวัสดุ เช่น เม็ดทรายหรือเม็ดดิน ออกแบบให้หว่าน โรย หรือหยอดตามตำแหน่ง ไม่ใช่นำไปละลายน้ำพ่น เว้นแต่ฉลากระบุเป็นสูตรเม็ดที่กระจายน้ำได้โดยเฉพาะ

### กลุ่มขนาดเม็ดที่เอกสารกล่าวถึง

| กลุ่ม              | คำย่อ |     ช่วงขนาดโดยประมาณ |
| ------------------ | ----- | --------------------: |
| เม็ดขนาดโต         | GG    | 2,000-6,000 ไมโครเมตร |
| เม็ดละเอียดปานกลาง | FG    |   300-2,500 ไมโครเมตร |
| เม็ดละเอียดมาก     | MG    |     100-600 ไมโครเมตร |

ช่วงขนาดอาจซ้อนกันและบางผลิตภัณฑ์อาจกำหนดขนาดด้วยหน่วย mesh จึงต้องอ้างอิงฉลากของผลิตภัณฑ์จริง

### จุดเด่น

- เม็ดมีขนาดใหญ่กว่าฝุ่น จึงปลิวเข้าสู่ระบบหายใจน้อยกว่า
- ใช้หว่าน โรย หรือหยอดได้โดยไม่ต้องใช้เครื่องพ่น
- สารออกฤทธิ์อาจค่อย ๆ ปลดปล่อยและออกฤทธิ์ได้นาน
- ใช้กับศัตรูในดินหรือสารดูดซึมบางชนิดได้ตามฉลาก

### ข้อควรระวัง

- สวมถุงมือและหน้ากากระหว่างหว่านหรือหยอด
- ห้ามใช้มือเปล่าหยิบสาร
- ห้ามหว่านใกล้แหล่งน้ำหรือบริเวณที่สัตว์เลี้ยงเข้าถึง
- กระจายตามอัตราต่อพื้นที่ ไม่กองรวมที่จุดเดียว
- อย่านำสูตร G หรือ GR ไปแช่น้ำเพื่อพ่น หากฉลากไม่ได้กำหนด

## 3. ฝุ่น ผง และเม็ดต่างกันอย่างไร

| ประเด็น        | ฝุ่นหรือผงใช้โดยตรง | เม็ด                                            |
| -------------- | ------------------- | ----------------------------------------------- |
| ขนาดอนุภาค     | เล็ก ฟุ้งง่าย       | ใหญ่กว่า ปลิวน้อยกว่า                           |
| วิธีใช้        | พ่นหรือเป่า         | หว่าน โรย หรือหยอด                              |
| น้ำ            | ไม่ผสมน้ำ           | ไม่ผสมน้ำ เว้นแต่เป็น WG/SG ซึ่งเป็นคนละสูตร    |
| ความเสี่ยงเด่น | สูดดมและการปลิว     | สัมผัสมือ การกินโดยสัตว์ และการกระจายผิดตำแหน่ง |
| อุปกรณ์        | เครื่องพ่นฝุ่นผง    | อาจใช้มือหรือเครื่องหว่านตามฉลาก                |

## 4. การใช้สารรม

สารรมออกฤทธิ์ในรูปก๊าซและสามารถแทรกซึมในพื้นที่ปิด วัสดุเก็บรักษา หรือดินตามการขึ้นทะเบียน เอกสารยกตัวอย่างสารที่เปลี่ยนเป็นก๊าซเมื่อสัมผัสความชื้น หรือสารที่ระเหยเป็นก๊าซที่อุณหภูมิห้อง

### ความเสี่ยงสูงมาก

- ก๊าซบางชนิดไม่มีสีและไม่มีกลิ่น
- เข้าสู่ร่างกายทางการหายใจอย่างรวดเร็ว
- อาจสะสมในพื้นที่ปิด
- การเปิดพื้นที่ก่อนครบเวลาระบายอากาศอาจทำให้เสียชีวิต
- ต้องมีเครื่องตรวจวัดและอุปกรณ์หายใจที่เหมาะสม

### ขอบเขตการปฏิบัติ

การรมสารไม่ควรทำโดยเกษตรกรทั่วไป ต้องดำเนินการโดยผู้ได้รับอนุญาตหรือผู้เชี่ยวชาญตามกฎหมาย ใช้ผลิตภัณฑ์ที่ขึ้นทะเบียน มีแผนกั้นพื้นที่ ป้ายเตือน การตรวจวัดก๊าซ การระบายอากาศ และแผนฉุกเฉิน

เอกสารนี้จึงอธิบายเฉพาะหลักการและความเสี่ยง ไม่ใช้เป็นคู่มือปฏิบัติการรมสาร

## 5. เครื่องพ่นหมอกไม่เท่ากับการรมทุกกรณี

เครื่องพ่นหมอกสร้างละอองขนาดเล็กหรือหมอกควันด้วยความร้อน ละอองสามารถลอยอยู่ในอากาศได้นาน แต่การใช้เครื่องพ่นหมอกต้องเป็นไปตามฉลาก พื้นที่ และเครื่องมือที่ออกแบบไว้ ไม่ควรนำตัวทำละลายหรือเชื้อเพลิงมาผสมเองโดยไม่มีคำแนะนำที่รับรอง

## 6. แบบตรวจสอบก่อนใช้สูตรไม่ผสมน้ำ

- [ ] อ่านแล้วว่าผลิตภัณฑ์ออกแบบให้ใช้โดยตรง
- [ ] สูตรตรงกับอุปกรณ์และวิธีใช้
- [ ] คำนวณอัตราต่อพื้นที่แล้ว
- [ ] สภาพลมและความชื้นเหมาะสม
- [ ] มีอุปกรณ์ป้องกันระบบหายใจและผิวหนัง
- [ ] กันคน สัตว์ อาหาร และแหล่งน้ำออกจากพื้นที่
- [ ] มีวิธีเก็บกวาดสารหกโดยไม่ทำให้ฟุ้ง
- [ ] ไม่ดัดแปลงสูตรเม็ดเป็นน้ำพ่นเอง

## หลักที่ควรจำ

> รหัสสูตรบอกวิธีใช้เบื้องต้น แต่ฉลากเป็นตัวกำหนดว่าผลิตภัณฑ์นั้นต้องพ่น หว่าน หยอด คลุกเมล็ด หรือใช้โดยผู้เชี่ยวชาญ
