# วิธีอ่านฉลากสารป้องกันกำจัดศัตรูพืชก่อนซื้อและก่อนใช้

ฉลากเป็นเอกสารกำกับผลิตภัณฑ์ที่บอกว่า สารนั้นคืออะไร ใช้กับพืชและศัตรูชนิดใด ใช้อัตราเท่าใด ต้องป้องกันตนเองอย่างไร และต้องทำอย่างไรเมื่อได้รับพิษ การอ่านเฉพาะชื่อการค้าหรือคำโฆษณาไม่เพียงพอ

## 1. ข้อมูลที่ต้องตรวจบนฉลาก

### ข้อมูลระบุตัวผลิตภัณฑ์

- ชื่อการค้า
- ชื่อสามัญของสารออกฤทธิ์
- เปอร์เซ็นต์สารออกฤทธิ์
- ชนิดสูตรผลิตภัณฑ์
- ปริมาณสุทธิ
- เลขทะเบียนวัตถุอันตราย
- ผู้ผลิตหรือผู้จำหน่าย
- วันเดือนปีผลิต
- เลขล็อตผลิต

### ข้อมูลการใช้

- พืชที่อนุญาตให้ใช้
- โรค แมลง ไร สัตว์ศัตรูพืช หรือวัชพืชเป้าหมาย
- อัตราใช้ต่อปริมาณน้ำหรือต่อพื้นที่
- ช่วงเวลาหรือระยะของศัตรูพืชที่เหมาะสม
- วิธีผสมและวิธีพ่น
- จำนวนครั้งหรือข้อจำกัดการใช้
- ความเข้ากันได้กับผลิตภัณฑ์อื่น
- ระยะเว้นก่อนเก็บเกี่ยว
- ข้อห้ามใช้

### ข้อมูลความปลอดภัย

- แถบสีและคำเตือนระดับอันตราย
- อุปกรณ์ป้องกันที่ต้องสวม
- อันตรายต่อสัตว์เลี้ยง ปลา ผึ้ง และสิ่งแวดล้อม
- อาการเมื่อได้รับพิษ
- วิธีปฐมพยาบาล
- คำแนะนำแก่แพทย์
- วิธีเก็บรักษา
- วิธีล้างอุปกรณ์และจัดการภาชนะ

## 2. แถบสีตามที่เอกสารอธิบาย

| แถบสี     | ความหมายตามเอกสาร |
| --------- | ----------------- |
| สีแดง     | พิษร้ายแรงมาก     |
| สีเหลือง  | พิษร้ายแรง        |
| สีน้ำเงิน | ควรระวัง          |

แถบสีต้องอ่านร่วมกับข้อความและสัญลักษณ์ ไม่ควรเลือกซื้อผลิตภัณฑ์จากสีเพียงอย่างเดียว

## 3. สัญลักษณ์ที่พบได้บนฉลาก

เอกสารแสดงตัวอย่างสัญลักษณ์สำหรับ

- เก็บมิดชิดให้พ้นมือเด็ก
- ชำระล้างหลังใช้
- อันตรายต่อสัตว์เลี้ยง
- อันตรายต่อปลาและสัตว์น้ำ ห้ามเทลงแหล่งน้ำ
- สวมอุปกรณ์ป้องกันดวงตา
- สวมอุปกรณ์ป้องกันจมูกและปาก
- สวมหน้ากากป้องกันไอพิษ
- สวมถุงมือ
- สวมผ้ากันเปื้อน
- สวมชุดป้องกันตลอดตัว
- สวมรองเท้าป้องกัน
- พิษร้ายแรงมาก พิษร้ายแรง หรืออันตราย

สัญลักษณ์ช่วยสื่อสารรวดเร็ว แต่ไม่แทนข้อความวิธีใช้และการปฐมพยาบาล

## 4. ลำดับอ่านฉลากแบบใช้งานจริง

### ก่อนซื้อ

1. ตรวจเลขทะเบียนวัตถุอันตราย
2. อ่านชื่อสามัญและความเข้มข้น
3. ตรวจว่าฉลากระบุพืชและศัตรูเป้าหมายตรงกับปัญหา
4. ดูขนาดบรรจุ อายุ และสภาพภาชนะ
5. เปรียบเทียบอัตราใช้ ระดับอันตราย และระยะเว้นก่อนเก็บเกี่ยว

### ก่อนผสม

1. อ่านอัตราใช้และปริมาณน้ำ
2. คำนวณตามพื้นที่จริง ไม่ใช้การกะประมาณ
3. ตรวจอุปกรณ์ป้องกันที่ฉลากกำหนด
4. อ่านข้อห้ามผสมและลำดับการผสม
5. เตรียมน้ำสะอาดและพื้นที่ผสมที่ไม่ใกล้อาหารหรือแหล่งน้ำ

### ก่อนพ่น

1. ตรวจเครื่องพ่นและหัวฉีด
2. ตรวจลมและสภาพอากาศ
3. กำหนดพื้นที่และปริมาณน้ำให้สัมพันธ์กับอัตรา
4. ตรวจระยะเว้นก่อนเก็บเกี่ยว
5. แจ้งผู้เกี่ยวข้องและกันคนกับสัตว์ออกจากพื้นที่

### หลังพ่น

1. ปฏิบัติตามข้อกำหนดการกลับเข้าพื้นที่
2. ล้างอุปกรณ์ในบริเวณที่เหมาะสม
3. จัดการน้ำล้างและภาชนะตามฉลาก
4. อาบน้ำ เปลี่ยนเสื้อผ้า และซักชุดทำงานแยก
5. บันทึกวันที่ พื้นที่ ชื่อสาร อัตรา และผู้พ่น

## 5. จุดที่มักอ่านผิด

| จุดที่มักผิด                               | ผลที่อาจเกิด                             |
| ------------------------------------------ | ---------------------------------------- |
| ดูชื่อการค้า แต่ไม่ดูชื่อสามัญ             | ใช้สารกลุ่มเดิมซ้ำโดยไม่รู้ตัว           |
| เห็นคำว่าใช้กำจัดแมลง จึงใช้กับแมลงทุกชนิด | ควบคุมไม่ได้และเพิ่มการดื้อสาร           |
| อ่านอัตราต่อน้ำ แต่ไม่คำนึงถึงพื้นที่      | ปริมาณสารต่อไร่สูงหรือต่ำเกินไป          |
| ไม่ดูสูตรและความเข้มข้น                    | ใช้อัตราของคนละผลิตภัณฑ์                 |
| ไม่ดูระยะเว้นก่อนเก็บเกี่ยว                | เสี่ยงสารตกค้าง                          |
| ไม่อ่านคำเตือนการผสม                       | เกิดตะกอน พืชเป็นพิษ หรือเพิ่มความเสี่ยง |
| เทสารใส่ขวดอื่นและทิ้งฉลาก                 | สูญเสียข้อมูลสำคัญและเสี่ยงกลืนกินผิด    |

## 6. แบบตรวจสอบสั้นก่อนใช้

- [ ] พืชตรง
- [ ] ศัตรูพืชตรง
- [ ] ชื่อสามัญและกลุ่มกลไกชัดเจน
- [ ] สูตรและความเข้มข้นตรงกับอัตราที่จะใช้
- [ ] คำนวณอัตราตามพื้นที่แล้ว
- [ ] มีอุปกรณ์ป้องกันครบ
- [ ] รักษาระยะเว้นก่อนเก็บเกี่ยวได้
- [ ] ไม่มีข้อห้ามหรือความเสี่ยงต่อพื้นที่ข้างเคียง
- [ ] ภาชนะและฉลากสมบูรณ์
- [ ] มีแผนจัดการสารที่เหลือและภาชนะใช้แล้ว

## หลักที่ควรจำ

> คำแนะนำในหนังสือช่วยเลือกแนวทาง ส่วนฉลากผลิตภัณฑ์ที่ขึ้นทะเบียนจริงเป็นข้อมูลที่ต้องตรวจสอบก่อนใช้ทุกครั้ง
