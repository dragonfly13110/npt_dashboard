# พิษเฉียบพลันและพิษเรื้อรังจากสารป้องกันกำจัดศัตรูพืช

ความเป็นพิษ หมายถึง ความสามารถของสารที่จะก่อให้เกิดอันตรายหรือการบาดเจ็บต่อสิ่งมีชีวิต ระดับอันตรายที่เกิดขึ้นจริงไม่ได้ขึ้นกับชนิดสารเพียงอย่างเดียว แต่ยังสัมพันธ์กับปริมาณที่ได้รับ เส้นทางที่เข้าสู่ร่างกาย ความถี่ ระยะเวลาสัมผัส และสูตรผลิตภัณฑ์

## 1. พิษเฉียบพลัน

พิษเฉียบพลันเกิดจากการได้รับสารในปริมาณค่อนข้างมากภายในช่วงเวลาสั้น อาจเกิดจากการสัมผัสเพียงครั้งเดียวหรือหลายครั้งในระยะใกล้กัน อาการมักปรากฏรวดเร็วหลังได้รับสาร

สถานการณ์ที่เสี่ยง ได้แก่

- สารเข้มข้นกระเด็นใส่ผิวหนังหรือดวงตาขณะเปิด ตวง หรือผสม
- สูดละอองหรือไอสารจำนวนมากจากการพ่นทวนลม
- ดื่มหรือกินสารจากภาชนะที่นำมาใส่ผิดประเภท
- เครื่องพ่นหรือสายยางรั่วจนสารเปื้อนร่างกาย
- เข้าไปในพื้นที่พ่นสารโดยไม่มีอุปกรณ์ป้องกัน

อาการที่เอกสารยกตัวอย่าง ได้แก่ อ่อนเพลีย วิงเวียนศีรษะ อาเจียน ปวดท้อง และท้องร่วง อาการจริงอาจแตกต่างตามชนิดสาร จึงต้องอ่านส่วนอาการพิษและการปฐมพยาบาลบนฉลากของผลิตภัณฑ์ทุกครั้ง

## 2. พิษเรื้อรัง

พิษเรื้อรังเกิดจากการได้รับสารครั้งละไม่มาก แต่ได้รับซ้ำเป็นเวลานาน อาการอาจไม่ปรากฏทันที ทำให้ผู้ใช้เข้าใจผิดว่าวิธีปฏิบัติเดิมปลอดภัย

สถานการณ์ที่อาจนำไปสู่การรับสัมผัสซ้ำ ได้แก่

- ผสมสารด้วยมือเปล่าหรือใช้ถุงมือที่ไม่ทนสาร
- สวมเสื้อผ้าที่ปนเปื้อนซ้ำโดยไม่ได้ซักแยก
- ใช้เครื่องพ่นที่รั่วเป็นประจำ
- พ่นสารบ่อยเกินความจำเป็น
- รับประทานอาหาร ดื่มน้ำ หรือสูบบุหรี่ระหว่างทำงาน
- เก็บสารใกล้อาหาร น้ำดื่ม หรือพื้นที่อยู่อาศัย
- ไม่อาบน้ำและเปลี่ยนเสื้อผ้าหลังเลิกงาน

## 3. ความแตกต่างที่ควรจำ

| ประเด็น           | พิษเฉียบพลัน                                                     | พิษเรื้อรัง                                                                |
| ----------------- | ---------------------------------------------------------------- | -------------------------------------------------------------------------- |
| รูปแบบการได้รับ   | ปริมาณมากในช่วงเวลาสั้น                                          | ปริมาณน้อยแต่ได้รับซ้ำเป็นเวลานาน                                          |
| การแสดงอาการ      | มักรวดเร็ว                                                       | อาจใช้เวลานานกว่าจะปรากฏ                                                   |
| ตัวอย่างเหตุการณ์ | สารหกใส่ สูดละอองมาก กลืนกิน                                     | พ่นซ้ำบ่อย อุปกรณ์รั่ว สัมผัสสารทุกครั้งโดยไม่ป้องกัน                      |
| วิธีลดความเสี่ยง  | หยุดสัมผัสทันที ล้างสารออก ปฏิบัติตามฉลาก และพบแพทย์เมื่อมีอาการ | ลดความถี่ ใช้ตามความจำเป็น ตรวจอุปกรณ์ สวมอุปกรณ์ป้องกัน และรักษาสุขอนามัย |

## 4. ความเป็นพิษไม่เท่ากับความเสี่ยง

สารที่มีความเป็นพิษสูงอาจมีความเสี่ยงต่ำลงได้เมื่อจัดการอย่างถูกต้องและไม่มีการสัมผัส ส่วนสารที่จัดว่ามีพิษต่ำก็ยังเกิดอันตรายได้ หากใช้ผิดอัตรา พ่นบ่อย ไม่สวมอุปกรณ์ป้องกัน หรือปล่อยให้ปนเปื้อนอาหารและสิ่งแวดล้อม

การประเมินความเสี่ยงจึงต้องดูพร้อมกันอย่างน้อย 5 เรื่อง

1. สารชนิดใดและสูตรใด
2. ได้รับเข้าสู่ร่างกายทางใด
3. ได้รับในปริมาณเท่าใด
4. ได้รับบ่อยและนานเพียงใด
5. มีมาตรการป้องกันเพียงพอหรือไม่

## 5. เมื่อสงสัยว่าได้รับพิษ

1. หยุดทำงานและออกจากบริเวณปนเปื้อน
2. หลีกเลี่ยงการสัมผัสซ้ำของผู้ช่วยเหลือ
3. ถอดเสื้อผ้าที่เปื้อนสารออก
4. ล้างผิวหนังด้วยน้ำและสบู่หลายครั้ง
5. อ่านคำแนะนำการปฐมพยาบาลบนฉลาก
6. นำฉลากหรือภาชนะผลิตภัณฑ์ไปด้วยเมื่อพบแพทย์
7. ไม่ทำให้อาเจียนเอง เว้นแต่ฉลากหรือบุคลากรทางการแพทย์สั่งให้ทำ

> ผู้ที่มีอาการหลังสัมผัสสารควรได้รับการประเมินจากบุคลากรทางการแพทย์ การจำแนกพิษเฉียบพลันหรือเรื้อรังในเอกสารนี้ใช้เพื่อการเรียนรู้ ไม่ใช้แทนการวินิจฉัย

## หลักที่ควรจำ

> อาการไม่เกิดทันที ไม่ได้หมายความว่าไม่มีการรับสารเข้าสู่ร่างกาย
