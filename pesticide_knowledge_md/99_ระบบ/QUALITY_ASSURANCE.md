# รายงานประกันคุณภาพโครงสร้างคลังองค์ความรู้ รุ่น 3.5

## ขอบเขตการตรวจ

การตรวจรอบนี้ครอบคลุมโครงสร้างไฟล์ Markdown ทั้งคลัง การอ่าน YAML front matter ตัวระบุ slug ชื่อเรื่อง ลิงก์ภายใน ความเชื่อมโยงกับหน้าต้นฉบับ MANIFEST และความสมบูรณ์ของไฟล์ ZIP

## รายการที่แก้ไขในรอบตรวจรับ

1. เพิ่ม slug, source_year และ last_reviewed ให้ไฟล์เดิม 5 ไฟล์ที่ข้อมูลกำกับยังไม่ครบ
2. เติม last_reviewed ให้ไฟล์พื้นฐาน กลไก และไฟล์ช่วงต้นที่ยังไม่มีวันทบทวน
3. เพิ่ม sections_count ในแม่แบบไฟล์องค์ความรู้
4. เพิ่มไฟล์ผู้จัดทำและคณะผู้วิจัยจากหน้าพิมพ์ 285
5. เพิ่มรายงานความครอบคลุม จุดตรวจสอบ และรายงานตรวจรับฉบับสมบูรณ์

## เกณฑ์ตรวจอัตโนมัติ

| รายการ                  | เกณฑ์ผ่าน                                                                     |
| ----------------------- | ----------------------------------------------------------------------------- |
| YAML                    | อ่านได้ทุกไฟล์ที่กำหนดให้มี front matter                                      |
| Metadata ของไฟล์เนื้อหา | มี title, slug, category, status, source_year, source_pages และ last_reviewed |
| Slug                    | ไม่ซ้ำทั้งคลัง                                                                |
| ชื่อเรื่อง              | ไม่ซ้ำแบบตรงตัวทั้งคลัง                                                       |
| ลิงก์ Markdown ภายใน    | ปลายทางมีอยู่จริง                                                             |
| Source coverage         | หน้าพิมพ์ที่มีเนื้อหาได้รับการเชื่อมโยงกับไฟล์องค์ความรู้                     |
| MANIFEST                | รายการตรงกับไฟล์จริงในคลัง                                                    |
| ZIP                     | เปิดอ่านได้และไม่พบสมาชิกเสียหายจาก testzip()                                 |

## ความหมายของการผ่านตรวจ

ผลผ่านในรายงานนี้หมายถึงคลังพร้อมใช้งานเชิงโครงสร้างสำหรับเว็บไซต์ การค้นหา และระบบ RAG ไม่ได้หมายถึงการรับรองว่าสารทุกผลิตภัณฑ์ยังขึ้นทะเบียน หรือคำแนะนำทุกอัตรายังคงใช้ได้ตามกฎหมายและฉลากปัจจุบัน

## ข้อจำกัดที่ยังต้องควบคุม

- คำแนะนำการใช้สารต้องตรวจฉลากและทะเบียนปัจจุบันก่อนใช้จริง
- บัญชีวัตถุอันตรายห้ามใช้เป็นการถอดจากเอกสารปี 2568 ไม่ใช่ฐานกฎหมายแบบเรียลไทม์
- จุดสะกดผิด หน่วยผิดสังเกต และตารางที่อ่านได้หลายแบบถูกเก็บเป็นธงตรวจสอบ ไม่แก้ต้นฉบับโดยไม่มีหลักฐาน
- การคำนวณอัตราต่อไร่จากอัตราต่อน้ำเป็นค่าที่ได้จากการคำนวณ ต้องสอบเทียบปริมาณน้ำจริงของเครื่องพ่น
