# คลังองค์ความรู้การใช้สารป้องกันกำจัดศัตรูพืชจากงานวิจัย ปี 2568

คลังองค์ความรู้นี้เรียบเรียงจากเอกสารวิชาการ **คำแนะนำการใช้สารป้องกันกำจัดศัตรูพืชจากงานวิจัย ปี 2568** ของกองวิจัยพัฒนาการอารักขาพืช กรมวิชาการเกษตร โดยปรับโครงสร้างเนื้อหาให้อ่านง่าย ค้นหาได้ และนำไปใช้เป็นฐานความรู้สำหรับเจ้าหน้าที่ส่งเสริมการเกษตร เกษตรกร ระบบเว็บไซต์ หรือระบบถามตอบด้วยปัญญาประดิษฐ์

## วัตถุประสงค์

1. อธิบายหลักการใช้สารป้องกันกำจัดศัตรูพืชอย่างมีประสิทธิภาพและปลอดภัย
2. แยกคำแนะนำตามชนิดพืช โรค แมลง ไร สัตว์ศัตรูพืช และวัชพืช
3. เชื่อมโยงชื่อสารกับกลุ่มกลไกการออกฤทธิ์ เพื่อช่วยวางแผนหมุนเวียนสาร
4. ลดความเสี่ยงจากการใช้ผิดชนิด ผิดอัตรา ผิดเวลา หรือใช้ซ้ำกลุ่มเดิมต่อเนื่อง
5. สนับสนุนการผลิตตามหลักเกษตรดีที่เหมาะสม

## ขอบเขตเนื้อหา

คลังนี้จะจัดทำเป็นตอนย่อย ครอบคลุมหัวข้อหลักดังต่อไปนี้

- ความปลอดภัยและการป้องกันอันตราย
- พิษและอันตรายของสารป้องกันกำจัดศัตรูพืช
- การเลือก การผสม และการใช้สาร
- สูตรผลิตภัณฑ์
- กลุ่มกลไกการออกฤทธิ์ FRAC, IRAC และ HRAC
- คำแนะนำสารป้องกันกำจัดโรคพืชรายพืช
- คำแนะนำสารกำจัดแมลง ไร และสัตว์ศัตรูพืชรายพืช
- คำแนะนำสารกำจัดวัชพืชรายพืช
- สถานการณ์ศัตรูพืชต้านทานสารและการหมุนเวียนสาร
- การใช้สารร่วมกับชีวภัณฑ์หรือสารธรรมชาติ
- เทคนิคการพ่นสารจากผลงานวิจัย
- รายชื่อวัตถุอันตรายที่ห้ามใช้ทางการเกษตร
- ภาคผนวก ดรรชนีชื่อสามัญ และตารางความเข้ากันได้ทางกายภาพ

## หลักการใช้ข้อมูล

- เอกสารแต่ละตอนเป็นการเรียบเรียงเพื่อการเรียนรู้ ไม่ใช่ฉลากผลิตภัณฑ์
- ก่อนใช้จริง ต้องตรวจสอบว่าผลิตภัณฑ์ยังขึ้นทะเบียนกับกรมวิชาการเกษตร และใช้ตามข้อความบนฉลาก
- ชื่อสามัญเดียวกันอาจมีหลายความเข้มข้นและหลายสูตร จึงห้ามนำอัตราของผลิตภัณฑ์หนึ่งไปใช้แทนอีกผลิตภัณฑ์หนึ่งโดยอัตโนมัติ
- สภาพการระบาด สภาพอากาศ ระยะการเจริญเติบโตของพืช และความต้านทานของศัตรูพืชในแต่ละพื้นที่ อาจทำให้ผลการควบคุมแตกต่างกัน
- ต้องสวมอุปกรณ์ป้องกันส่วนบุคคลที่เหมาะสมทุกครั้ง
- ต้องปฏิบัติตามระยะเว้นก่อนเก็บเกี่ยวที่ระบุในคำแนะนำและฉลาก

## รูปแบบไฟล์

ไฟล์องค์ความรู้ใช้ Markdown และมีส่วนข้อมูลกำกับด้านบน เพื่อให้สามารถนำไปใช้กับเว็บไซต์ ระบบจัดการเนื้อหา และฐานความรู้ได้ง่าย

## สถานะการจัดทำ

การแปลงเอกสารต้นฉบับเป็นคลังองค์ความรู้ครบตามหน้าที่มีเนื้อหาแล้ว และในรุ่น 3.6 ได้เพิ่มชุดส่งออก JSONL/CSV โครงสร้าง pgvector คำสั่งระบบ และชุดทดสอบสำหรับนำไปทำ RAG

## แหล่งอ้างอิงหลัก

ศรีจำนรรจ์ ศรีจันทรา และพฤทธิชาติ ปุญวัฒโท. 2568. _เอกสารวิชาการ คำแนะนำการใช้สารป้องกันกำจัดศัตรูพืชจากงานวิจัย ปี 2568_. กรมวิชาการเกษตร.

## ชุดส่งออกสำหรับระบบ RAG รุ่น 3.6

ดูคู่มือในโฟลเดอร์ `12_ชุดข้อมูลสำหรับระบบ_RAG` และไฟล์พร้อมนำเข้าใน `99_ระบบ/RAG_EXPORT` ชุดส่งออกนี้ยังคงข้อกำหนดให้ตรวจฉลาก ทะเบียน PHI และกฎหมายปัจจุบันก่อนใช้จริง
