# FRAC คืออะไร และอ่านรหัสสารป้องกันกำจัดโรคพืชอย่างไร

FRAC เป็นระบบจำแนกสารป้องกันกำจัดโรคพืชตามตำแหน่งและกระบวนการออกฤทธิ์ต่อเชื้อสาเหตุโรค ใช้ช่วยวางแผนสลับกลุ่มและลดแรงคัดเลือกที่ทำให้เชื้อพัฒนาความต้านทาน

เอกสารปี 2568 แบ่งกลไกเป็นหมวดอักษร A ถึง I รวมทั้งกลุ่ม P, U, M และ BM พร้อมแสดงเลขรหัสและการประเมินความเสี่ยงต่อความต้านทานของแต่ละกลุ่ม

## 1. โครงสร้างรหัส FRAC

ตัวอย่าง

| สิ่งที่เห็น | ความหมาย                                                               |
| ----------- | ---------------------------------------------------------------------- |
| FRAC 3      | กลุ่มยับยั้ง C14-demethylase ในการสังเคราะห์สเตอรอล                    |
| FRAC 7      | กลุ่มยับยั้ง succinate dehydrogenase ที่ complex II                    |
| FRAC 11     | กลุ่มยับยั้งการหายใจที่ complex III ตำแหน่ง Qo                         |
| FRAC M03    | สารสัมผัสหลายตำแหน่ง กลุ่ม dithiocarbamates และสารใกล้เคียง            |
| FRAC P07    | phosphonates ที่เกี่ยวข้องกับการกระตุ้นการป้องกันของพืชและมีผลต่อเชื้อ |
| FRAC BM02   | จุลชีพหรือผลิตผลจุลชีพที่มีกลไกหลายแบบ                                 |

เลขรหัสไม่ได้เรียงตามความแรง และเลขที่ขาดหายไม่ได้หมายความว่าเอกสารไม่สมบูรณ์ เพราะ FRAC ปรับปรุงและย้ายกลุ่มตามข้อมูลวิทยาศาสตร์ใหม่เป็นระยะ

## 2. กลุ่มออกฤทธิ์ตำแหน่งเดียวกับหลายตำแหน่ง

### กลุ่มออกฤทธิ์เฉพาะตำแหน่ง

สารหลายกลุ่มใน A ถึง I มีเป้าหมายจำเพาะ เช่น เอนไซม์หนึ่งชนิดหรือจุดหนึ่งในระบบหายใจ ข้อดีคือมีประสิทธิภาพสูงและมักออกฤทธิ์เป็นระบบหรือเฉพาะเจาะจง แต่เมื่อใช้ซ้ำมาก ความต้านทานอาจพัฒนาได้เร็ว โดยเฉพาะกลุ่มที่เอกสารระบุความเสี่ยงสูง เช่น

- FRAC 1 กลุ่ม benzimidazoles และ thiophanates
- FRAC 4 กลุ่ม acylalanines และสารใกล้เคียง
- FRAC 11 กลุ่มที่ออกฤทธิ์ต่อ complex III ตำแหน่ง Qo
- FRAC 25 streptomycin
- FRAC 53 pyridachlometyl

### กลุ่มออกฤทธิ์หลายตำแหน่ง

รหัส M01 ถึง M12 เป็นสารสัมผัสที่มีผลหลายตำแหน่ง เอกสารประเมินความเสี่ยงต่อความต้านทานโดยทั่วไปว่าอยู่ในระดับต่ำ ตัวอย่าง ได้แก่ ทองแดง กำมะถัน mancozeb, captan, chlorothalonil และ dithianon

คำว่าเสี่ยงต่ำไม่ได้แปลว่าใช้ได้ถี่โดยไม่มีข้อจำกัด ยังต้องใช้ตามฉลาก คำนึงถึงสารตกค้าง ผลกระทบต่อพืช ผู้ใช้ และสิ่งแวดล้อม

## 4. การอ่านสูตรผสม

ตัวอย่างเชิงหลักการ หากผลิตภัณฑ์มีสาร FRAC 7/11 หมายความว่ามีองค์ประกอบที่ออกฤทธิ์สองกลุ่ม การบันทึกต้องเขียนทั้ง 7 และ 11 ไม่ควรบันทึกเพียงกลุ่มใดกลุ่มหนึ่ง

การใช้สูตรผสม 7/11 แล้วตามด้วยสารเดี่ยวกลุ่ม 11 ยังถือว่ามีการใช้แรงคัดเลือกต่อกลุ่ม 11 ต่อเนื่อง

## 5. ความเสี่ยงต่อความต้านทานแปลอย่างไร

| ระดับในเอกสาร | การตีความเพื่อวางแผน                                                      |
| ------------- | ------------------------------------------------------------------------- |
| สูง           | จำกัดจำนวนครั้ง หลีกเลี่ยงใช้ต่อเนื่อง และต้องวางแผนหมุนเวียนอย่างเข้มงวด |
| ปานกลาง-สูง   | ใช้ตามโปรแกรมที่สลับกลุ่มและเฝ้าระวังประสิทธิภาพ                          |
| ปานกลาง       | ยังต้องหมุนเวียนและหลีกเลี่ยงการใช้ซ้ำโดยไม่จำเป็น                        |
| ต่ำ-ปานกลาง   | ความเสี่ยงน้อยลง แต่ไม่ควรใช้จนกลายเป็นกลุ่มหลักเพียงกลุ่มเดียว           |
| ต่ำ           | เหมาะเป็นส่วนหนึ่งของโปรแกรมผสมผสาน แต่ยังต้องปฏิบัติตามฉลาก              |
| ไม่มีข้อมูล   | ไม่ได้หมายถึงไม่มีความเสี่ยง เพียงยังมีข้อมูลไม่เพียงพอ                   |

## 6. ขั้นตอนใช้รหัส FRAC ในแปลง

1. วินิจฉัยโรคและเชื้อสาเหตุให้ใกล้เคียงที่สุด
2. ตรวจว่าผลิตภัณฑ์ขึ้นทะเบียนกับพืชและโรคเป้าหมาย
3. จดชื่อสามัญ สูตร ความเข้มข้น และรหัส FRAC ทุกองค์ประกอบ
4. ตรวจประวัติการใช้ในรอบการระบาดเดียวกัน
5. เลือกครั้งถัดไปจากคนละรหัสที่มีประสิทธิภาพต่อโรคนั้น
6. ใช้วิธีลดแหล่งเชื้อ พันธุ์ต้านทาน การจัดการความชื้น และสุขอนามัยแปลงร่วมด้วย
7. ประเมินผลหลังใช้ หากประสิทธิภาพลดลงต้องตรวจสาเหตุอื่นก่อนเพิ่มอัตรา

## 7. ข้อผิดพลาดที่พบบ่อย

- เปลี่ยนจาก azoxystrobin เป็น pyraclostrobin แล้วคิดว่าเปลี่ยนกลุ่ม ทั้งสองอยู่ FRAC 11
- เปลี่ยนยี่ห้อแต่ชื่อสามัญและรหัสเดิม
- ใช้สูตรผสมแล้วบันทึกเพียงสารตัวแรก
- เห็นคำว่า multi-site แล้วใช้ถี่เกินฉลาก
- เห็นชื่อชีวภัณฑ์แล้วถือว่าทุกสายพันธุ์ใช้แทนกันได้

## ข้อควรระวัง

รายชื่อในตาราง FRAC เป็นการจำแนกกลไก ไม่ใช่บัญชีอนุญาตใช้ ต้องตรวจสอบสถานะทะเบียนและฉลากปัจจุบันก่อนใช้จริง
