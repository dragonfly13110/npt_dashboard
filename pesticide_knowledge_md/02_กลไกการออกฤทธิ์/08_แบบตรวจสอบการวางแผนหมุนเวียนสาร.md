# แบบตรวจสอบการวางแผนหมุนเวียนสารป้องกันกำจัดศัตรูพืช

แบบตรวจสอบนี้ใช้ก่อนอนุมัติคำแนะนำหรือก่อนจัดโปรแกรมสารในแปลง จุดประสงค์คือป้องกันการเปลี่ยนเฉพาะชื่อการค้าแต่ยังใช้กลุ่มกลไกเดิม ลดการพ่นเกินจำเป็น และช่วยบันทึกหลักฐานเมื่อประสิทธิภาพลดลง

## ส่วนที่ 1 ยืนยันปัญหา

- [ ] ระบุพืช พันธุ์ อายุ และระยะการเจริญเติบโตแล้ว
- [ ] ระบุศัตรูพืชถึงระดับที่ใช้ตัดสินใจเลือกสารได้
- [ ] ตรวจว่าความเสียหายไม่ได้เกิดจากธาตุอาหาร น้ำ สารพิษต่อพืช หรือสภาพอากาศ
- [ ] สำรวจระดับการระบาดและตำแหน่งกระจายในแปลงแล้ว
- [ ] พิจารณาว่าถึงเกณฑ์ที่จำเป็นต้องใช้สารหรือยัง

## ส่วนที่ 2 ตรวจผลิตภัณฑ์

- [ ] ผลิตภัณฑ์ยังขึ้นทะเบียน
- [ ] ฉลากระบุพืชและศัตรูเป้าหมายตรงกับงาน
- [ ] ตรวจชื่อสามัญครบทุกองค์ประกอบ
- [ ] ตรวจสูตรและความเข้มข้น
- [ ] ตรวจอัตราใช้ ปริมาณน้ำ วิธีพ่น และระยะเว้นก่อนเก็บเกี่ยว
- [ ] ตรวจข้อห้ามเกี่ยวกับผึ้ง แหล่งน้ำ พืชข้างเคียง และผู้ปฏิบัติงาน

## ส่วนที่ 3 บันทึกรหัสกลไก

| ผลิตภัณฑ์ | ชื่อสามัญทั้งหมด | ระบบ           | รหัสกลุ่ม | ใช้ล่าสุดเมื่อไร | จำนวนครั้งในรอบนี้ |
| --------- | ---------------- | -------------- | --------- | ---------------- | ------------------ |
|           |                  | FRAC/IRAC/HRAC |           |                  |                    |

- [ ] สูตรผสมบันทึกรหัสครบทุกสาร
- [ ] การผสมในถังบันทึกทุกผลิตภัณฑ์
- [ ] ไม่สลับจากสูตรผสมไปสารเดี่ยวที่เป็นองค์ประกอบเดิมทันทีโดยไม่ประเมินแรงคัดเลือก
- [ ] กลุ่มที่เลือกครั้งถัดไปต่างจากกลุ่มก่อนหน้าและยังมีประสิทธิภาพต่อเป้าหมาย

## ส่วนที่ 4 ประเมินความเสี่ยงการใช้ซ้ำ

ให้คะแนนเบื้องต้น

| ปัจจัย             | 0 คะแนน           | 1 คะแนน                     | 2 คะแนน         |
| ------------------ | ----------------- | --------------------------- | --------------- |
| การใช้กลุ่มเดิม    | ไม่ใช้ซ้ำ         | เคยใช้ก่อนหน้าแต่มีช่วงคั่น | ใช้ต่อเนื่อง    |
| จำนวนครั้งในฤดู    | 1 ครั้ง           | 2 ครั้ง                     | 3 ครั้งขึ้นไป   |
| ประวัติประสิทธิภาพ | ปกติ              | เริ่มแปรปรวน                | ลดลงชัดเจน      |
| วิธีอื่นร่วม       | หลายวิธี          | มีบางวิธี                   | พึ่งสารเป็นหลัก |
| ความถูกต้องการพ่น  | สอบเทียบและคลุมดี | มีข้อสงสัยบางส่วน           | ไม่ได้สอบเทียบ  |

การตีความเพื่อทบทวนงาน

- 0-3 คะแนน: ความเสี่ยงจากรูปแบบการใช้ต่ำกว่า แต่ยังต้องใช้ตามฉลาก
- 4-6 คะแนน: ต้องทบทวนกลุ่มและวิธีอื่นร่วมก่อนพ่น
- 7-10 คะแนน: ไม่ควรเดินหน้าด้วยโปรแกรมเดิมจนกว่าจะตรวจสาเหตุและปรับแผน

คะแนนนี้เป็นเครื่องมือคัดกรองภายใน ไม่ใช่มาตรฐานทางกฎหมายหรือการยืนยันความต้านทาน

## ส่วนที่ 5 ตรวจการผสม

- [ ] มีเหตุผลชัดเจนว่าทุกสารจำเป็นต่อศัตรูเป้าหมาย
- [ ] ฉลากหรือข้อมูลวิชาการรองรับ
- [ ] ทดสอบการเข้ากันได้ในภาชนะเล็กแล้ว
- [ ] ไม่มีตะกอน ก้อน ความร้อน ฟอง หรือการแยกชั้นผิดปกติ
- [ ] ไม่ผสมชีวภัณฑ์กับสารที่อาจทำลายเชื้อโดยไม่มีข้อมูลรองรับ
- [ ] คำนวณอัตราต่อพื้นที่ ไม่ใช้หลักคิดอัตราต่อถังอย่างเดียว

## ส่วนที่ 6 วิธีลดจำนวนครั้งการใช้สาร

เลือกใช้ให้เหมาะกับปัญหา

- [ ] พันธุ์ต้านทานหรือเมล็ดพันธุ์สะอาด
- [ ] ทำลายแหล่งสะสมโรค แมลง หรือเมล็ดวัชพืช
- [ ] ปรับระยะปลูก การระบายอากาศ น้ำ และธาตุอาหาร
- [ ] ใช้กับดัก การสำรวจ และพยากรณ์การระบาด
- [ ] รักษาศัตรูธรรมชาติ
- [ ] ใช้ชีวภัณฑ์หรือสารธรรมชาติที่ขึ้นทะเบียนและเหมาะกับระยะ
- [ ] ถอน ตัด ไถกลบ หรือจัดการวัชพืชก่อนสร้างเมล็ด
- [ ] หมุนเวียนพืชและกลไกการจัดการ

## ส่วนที่ 7 บันทึกหลังใช้

| ------------- | ------------ | --------------- | ----------------- | ------------ | ---------- | ------------- | ------- |
|               |              |                 |                   |              |            |               |         |

## เมื่อประสิทธิภาพต่ำ ห้ามทำสิ่งใดก่อนตรวจสาเหตุ

- ห้ามเพิ่มอัตราเอง
- ห้ามพ่นกลุ่มเดิมซ้ำทันทีเพราะคิดว่าสารยังไม่แรงพอ
- ห้ามผสมสารหลายชนิดแบบไม่มีข้อมูล
- ห้ามสรุปว่าเกิดความต้านทานจากการพ่นครั้งเดียว

ควรตรวจ

1. ระบุชนิดศัตรูถูกหรือไม่
2. ระยะศัตรูไวต่อสารหรือไม่
3. อัตราและปริมาณน้ำถูกหรือไม่
4. เครื่องพ่นและหัวฉีดทำงานปกติหรือไม่
5. ฝน ลม อุณหภูมิ และความชื้นเหมาะหรือไม่
6. สารเสื่อมสภาพหรือเก็บรักษาผิดหรือไม่
7. มีประวัติใช้กลุ่มเดิมซ้ำหรือไม่
8. มีตัวอย่างที่ควรส่งตรวจความต้านทานหรือไม่

```text
พืช:
ศัตรูพืช:
พื้นที่และระยะระบาด:
สารที่ใช้ 3 ครั้งล่าสุด:
1) ชื่อสามัญ / รหัส / วันที่ / ผล
2) ชื่อสามัญ / รหัส / วันที่ / ผล
3) ชื่อสามัญ / รหัส / วันที่ / ผล
สภาพอากาศและวิธีพ่น:
ข้อสงสัยหลัก:
รูปถ่ายหรือจุดพิกัด:
```

## หลักที่ควรจำ

> โปรแกรมที่ดีต้องเปลี่ยนทั้งกลไก ลดจำนวนครั้ง และจัดการสาเหตุของการระบาด ไม่ใช่เปลี่ยนชื่อขวดไปเรื่อย ๆ

## แหล่งอ้างอิง

ประยุกต์จากหลักการจำแนกกลุ่ม FRAC, IRAC และ HRAC ใน ศรีจำนรรจ์ ศรีจันทรา และพฤทธิชาติ ปุญวัฒโท. 2568. _เอกสารวิชาการ คำแนะนำการใช้สารป้องกันกำจัดศัตรูพืชจากงานวิจัย ปี 2568_. กรมวิชาการเกษตร. หน้า ii และ 22-46.
