# มะม่วง: การใช้อากาศยานไร้คนขับพ่นสารป้องกันกำจัดเพลี้ยไฟ

ต้นฉบับเสนอ UAV เป็นทางเลือกเพื่อลดแรงงาน เวลา และการสัมผัสสารของผู้พ่น โดยกำหนดอัตราน้ำต่อต้นตามขนาดทรงพุ่ม พร้อมระยะบินเหนือทรงพุ่มและแนวกันชนพื้นที่นอกเป้าหมาย

## 1. ค่าการพ่นตามต้นฉบับ

| เส้นผ่านศูนย์กลางทรงพุ่มเฉลี่ย | อัตราน้ำต่อต้น |
| -----------------------------: | -------------: |
|                       3-4 เมตร |    200 มล./ต้น |
|                       5-6 เมตร |    300 มล./ต้น |

| พารามิเตอร์                 | ค่าตามต้นฉบับ                   |
| --------------------------- | ------------------------------- |
| ความสูงบิน                  | 2 เมตรเหนือทรงพุ่ม              |
| ความเร็วลม                  | ไม่เกิน 3 กิโลเมตรต่อชั่วโมง    |
| ระยะจากพื้นที่นอกเป้าหมาย   | ไม่น้อยกว่า 18 เมตร             |
| เวลาโดยประมาณต่อไร่         | UAV 3-4 นาที                    |
| เวลาเปรียบเทียบการพ่นด้วยคน | 30-40 นาทีต่อไร่                |
| การประหยัดเวลา              | ประมาณ 8-11 เท่า ตามผลในต้นฉบับ |

## 2. การวางแผนและสอบเทียบ UAV

1. สำรวจจำนวนต้น ระยะปลูก และขนาดทรงพุ่มจริงก่อนคำนวณน้ำรวม
2. แบ่งต้นเป็นกลุ่มตามขนาดทรงพุ่มเพื่อหลีกเลี่ยงการใช้อัตราเดียวกับทุกต้น
3. ตรวจชนิดหัวพ่น ขนาดละออง อัตราการไหล ความเร็วบิน และความกว้างแนวพ่น
4. ใช้น้ำสะอาดทดสอบการครอบคลุมบนช่อดอก ยอดอ่อน และด้านในทรงพุ่ม
5. กำหนดเขตกันชนจากคน บ้าน ถนน แหล่งน้ำ สัตว์เลี้ยง และพืชข้างเคียง
6. คำนวณสารตามอัตราต่อพื้นที่หรือต่อต้นที่ฉลากอนุญาต ห้ามลดน้ำแล้วเพิ่มความเข้มข้นโดยไม่ตรวจฉลาก

### PHI

การพ่นด้วย UAV ไม่เปลี่ยน PHI ต้องยึดฉลากของสารที่เลือกใช้กับมะม่วงและเพลี้ยไฟ รวมถึงข้อกำหนดตลาดปลายทาง

## 3. ประโยชน์ ข้อจำกัด และเงื่อนไขใช้จริง

ต้นฉบับระบุประโยชน์ด้านความเร็ว ลดแรงงาน และลดการสัมผัสสารโดยตรงของผู้พ่น อย่างไรก็ตามประสิทธิภาพขึ้นกับการเข้าถึงช่อดอกและทรงพุ่ม การกระจายละออง สภาพลม และการสอบเทียบ

## จุดที่ต้องระวังและตรวจสอบ

- การลดอัตราน้ำหรือเปลี่ยนเครื่องพ่นไม่อนุญาตให้เพิ่มอัตราสารต่อพื้นที่เกินฉลาก ต้องสอบเทียบเครื่อง ควบคุมการปลิว และสวมอุปกรณ์ป้องกันตามความเสี่ยงของผลิตภัณฑ์จริง
- ค่าลม “ไม่เกิน 3 กิโลเมตรต่อชั่วโมง” เป็นค่าตามต้นฉบับ ต้องตรวจมาตรฐานการบิน คู่มือ UAV และฉลากผลิตภัณฑ์ปัจจุบันร่วมด้วย
- ระยะกันชน 18 เมตรเป็นผลจากงานวิจัยในเงื่อนไขที่กำหนด ไม่ควรลดระยะเมื่อสภาพลม หัวพ่น หรือขนาดละอองต่างจากงานวิจัย
- ต้องปฏิบัติตามกฎหมายการบิน การขึ้นทะเบียนผู้บังคับอากาศยาน และข้อกำหนดการใช้วัตถุอันตรายปัจจุบัน
- ห้ามบินเหนือคน สัตว์ บ้าน ถนน หรือพื้นที่ที่ไม่เกี่ยวข้องกับการพ่น
- ต้องตรวจทะเบียนว่าผลิตภัณฑ์นั้นอนุญาตใช้กับมะม่วง เพลี้ยไฟ และวิธีพ่นที่เลือก
- ข้อมูลนี้ใช้เพื่อการเรียนรู้และค้นข้อมูล ไม่ใช้แทนทะเบียน ฉลากผลิตภัณฑ์ ประกาศทางกฎหมาย หรือข้อกำหนดปัจจุบัน ก่อนใช้หรืออ้างอิงทางราชการต้องตรวจสอบแหล่งข้อมูลที่มีผลใช้บังคับล่าสุดทุกครั้ง
