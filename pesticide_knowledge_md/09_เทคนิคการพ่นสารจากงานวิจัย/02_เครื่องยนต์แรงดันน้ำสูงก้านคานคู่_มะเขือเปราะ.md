# มะเขือเปราะ: เครื่องยนต์พ่นสารแรงดันน้ำสูงพร้อมก้านฉีดคานคู่แนวตั้ง

ต้นฉบับอธิบายการดัดแปลงเครื่องยนต์พ่นสารสะพายหลังแบบแรงดันน้ำสูง โดยประกอบก้านฉีดเป็นคานคู่แนวตั้งสองด้านและใช้หัวฉีดแบบพัด เพื่อให้ละอองเข้าถึงด้านบน กลาง ล่าง รวมทั้งด้านบนใบและใต้ใบของทรงพุ่มมะเขือเปราะได้ทั่วถึงขึ้น

## 1. ลักษณะเครื่องและการดัดแปลง

| รายการ             | รายละเอียดตามต้นฉบับ                        |
| ------------------ | ------------------------------------------- |
| ชนิดเครื่อง        | เครื่องยนต์พ่นสารสะพายหลังแบบแรงดันน้ำสูง   |
| ชื่ออังกฤษ         | Motorised knapsack sprayer                  |
| ความจุถังโดยทั่วไป | 12-25 ลิตร                                  |
| เครื่องยนต์        | ส่วนใหญ่เป็นเครื่องยนต์ 2 จังหวะ ใช้ขับปั๊ม |
| หัวฉีดทั่วไป       | หัวกรวยกลวงหรือหัวพัด จำนวน 1-4 หัว         |
| การดัดแปลง         | ก้านฉีดคานคู่แนวตั้งสองด้าน                 |
| หัวฉีดในงานวิจัย   | หัวฉีดแบบพัด                                |
| พืชเป้าหมาย        | มะเขือเปราะ                                 |

คานคู่แนวตั้งช่วยพ่นครอบคลุมทั้งต้นโดยไม่ต้องส่ายหัวฉีดไปมา และสามารถเลือกเปิดหรือปรับตำแหน่งหัวฉีดให้เหมาะกับความสูงของพืช

## 2. หลักการใช้งานและการสอบเทียบ

ก่อนพ่นต้องสอบเทียบอัตราการไหลของหัวฉีดทุกหัว ความดัน ความเร็วเดิน ระยะห่างจากทรงพุ่ม และปริมาณน้ำต่อพื้นที่จริง หัวฉีดด้านซ้ายและขวาควรมีอัตราการไหลใกล้เคียงกัน และแนวพ่นต้องซ้อนทับพอเหมาะโดยไม่ทำให้เกิดการไหลรวมเป็นหยด

### ขั้นตอนตรวจภาคสนาม

1. ตรวจการรั่วของสาย ก๊อก ข้อต่อ และหัวฉีด
2. เติมน้ำสะอาดและวัดอัตราการไหลของหัวฉีดแต่ละหัวในช่วงเวลาเท่ากัน
3. เปลี่ยนหัวฉีดที่สึกหรือมีอัตราการไหลแตกต่างจากหัวอื่นอย่างชัดเจน
4. ทดลองเดินด้วยน้ำเปล่าในแปลงจริง แล้วคำนวณน้ำต่อไร่
5. ปรับจำนวนหัวฉีดตามความสูงและความหนาแน่นของทรงพุ่ม
6. คำนวณสารจากอัตราต่อพื้นที่ตามฉลาก ไม่คำนวณจากความจุถังเพียงอย่างเดียว

### PHI

เทคนิคก้านฉีดไม่เปลี่ยน PHI ของสาร ต้องยึดระยะเว้นก่อนเก็บเกี่ยวของผลิตภัณฑ์จริงตามฉลาก

## 3. จุดเด่น ข้อจำกัด และการลดความเสี่ยง

ต้นฉบับระบุจุดเด่นว่า ช่วยประหยัดเวลาเดินพ่น ละอองเข้าสู่ทรงพุ่มได้ทั่วถึง และเพิ่มโอกาสควบคุมแมลงที่อาศัยทั้งบนใบและใต้ใบ

ข้อจำกัดสำคัญคือเครื่องชนิดนี้ใช้น้ำมาก ต้องเติมน้ำหลายครั้งในพื้นที่กว้าง และน้ำหนักเครื่องรวมของเหลวอาจเพิ่มภาระต่อผู้ปฏิบัติงาน การใช้แรงดันสูงเกินไปยังเพิ่มละอองขนาดเล็กและการปลิวออกนอกเป้าหมาย

## จุดที่ต้องระวังและตรวจสอบ

- การลดอัตราน้ำหรือเปลี่ยนเครื่องพ่นไม่อนุญาตให้เพิ่มอัตราสารต่อพื้นที่เกินฉลาก ต้องสอบเทียบเครื่อง ควบคุมการปลิว และสวมอุปกรณ์ป้องกันตามความเสี่ยงของผลิตภัณฑ์จริง
- ต้องหยุดพ่นเมื่อหัวฉีดอุดตันก่อนทำความสะอาด ห้ามใช้ปากเป่าหัวฉีด
- ตรวจความสม่ำเสมอของหัวฉีดทั้งสองด้าน เพราะหัวที่สึกทำให้อัตราสารต่อพื้นที่ไม่เท่ากัน
- หลีกเลี่ยงพ่นเข้าหาผู้ปฏิบัติงาน บ้าน แหล่งน้ำ สัตว์ และพืชอ่อนไหว
- ค่าความจุถังและรายละเอียดเครื่องเป็นข้อมูลทั่วไปในต้นฉบับ ต้องตรวจคู่มือรุ่นที่ใช้จริง
- ข้อมูลนี้ใช้เพื่อการเรียนรู้และค้นข้อมูล ไม่ใช้แทนทะเบียน ฉลากผลิตภัณฑ์ ประกาศทางกฎหมาย หรือข้อกำหนดปัจจุบัน ก่อนใช้หรืออ้างอิงทางราชการต้องตรวจสอบแหล่งข้อมูลที่มีผลใช้บังคับล่าสุดทุกครั้ง
