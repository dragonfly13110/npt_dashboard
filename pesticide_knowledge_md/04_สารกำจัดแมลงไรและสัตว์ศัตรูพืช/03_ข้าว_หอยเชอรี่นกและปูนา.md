# คำแนะนำการจัดการหอยเชอรี่ นก และปูนาในนาข้าว

## หอยเชอรี่หรือหอยโข่งอเมริกาใต้

ชื่อวิทยาศาสตร์ _Pomacea canaliculata_

### นิโคลซาไมด์-โอลามีน

| รายการ           | ข้อมูล                                          |
| ---------------- | ----------------------------------------------- |
| ชื่อสามัญ        | นิโคลซาไมด์-โอลามีน หรือ นิโคลซาไมด์เอทาโนลามีน |
| สูตร             | 83.1% WP                                        |
| ระดับพิษและ LD50 | ไม่มีพิษเฉียบพลัน (5,000)                       |
| อัตรา            | 50 กรัม/ไร่ หรือ 25 กรัม/น้ำ 20 ลิตร            |
| วิธีใช้          | ผสมน้ำพ่นทั่วนา เน้นแอ่งหรือบริเวณที่มีหอยมาก   |

ต้นฉบับกำหนดให้ใช้สารกำจัดหอยร่วมกับตาข่ายถี่กั้นทางน้ำเข้าออก และต้องมีน้ำในนาเพื่อให้หอยเปิดฝาและมีกิจกรรม

### เมทัลดีไฮด์และกากเมล็ดชา

| รายการ                    | สูตร        | ระดับพิษและ LD50  |          อัตรา |
| ------------------------- | ----------- | ----------------- | -------------: |
| เมทัลดีไฮด์ (metaldehyde) | 5% GB       | ปานกลาง (283)     |   500 กรัม/ไร่ |
| เมทัลดีไฮด์ (metaldehyde) | 3.5% GR     | ไม่แสดงแยกในตาราง | 2,000 กรัม/ไร่ |
| กากเมล็ดชา (saponin)      | 10% saponin | ไม่ระบุ           | 3,000 กรัม/ไร่ |

### วิธีใช้ในกลุ่มรายการ

- หว่านลงน้ำให้ทั่ว เน้นแอ่งหรือบริเวณมีหอยมาก
- รักษาระดับน้ำประมาณ 5 เซนติเมตรต่อเนื่องอย่างน้อย 3 วัน
- ต้นฉบับระบุให้ใช้สารกำจัดหอยเพียงครั้งเดียวต่อฤดูปลูก

ตารางจัดข้อความวิธีใช้รวมหลายแถว จึงต้องตรวจฉลากของแต่ละผลิตภัณฑ์เพื่อยืนยันว่าวิธีดังกล่าวใช้กับรายการนั้นโดยตรง

## การจัดการนกในนาข้าว

นกที่ต้นฉบับระบุ ได้แก่ นกกระติ๊ดขี้หมู นกกระติ๊ดตะโพกขาว นกกระจอกตาล นกกระจาบธรรมดา และนกกระจาบอกลาย

แนวทางที่ต้นฉบับระบุ

1. จัดการแหล่งอยู่อาศัยหรือทำรังใกล้แปลง
2. ใช้ตาข่าย โดยแปลงขนาดเล็กอาจคลุมทั้งแปลง
3. ใช้เสียงไล่
4. ใช้วัสดุสะท้อนแสงขึงทั่วแปลง

ควรวางวิธีให้ลดการทำอันตรายต่อสัตว์ที่ไม่ใช่เป้าหมายและไม่สร้างความเดือดร้อนแก่พื้นที่รอบข้าง

## การจัดการปูนา

ชนิดที่ต้นฉบับระบุรวม _Sayamia bangkokensis_, _S. germaini_, _S. sexpunctata_ และ _Esanthelphusa dugasti_

แนวทางที่ต้นฉบับระบุ

1. กำจัดแหล่งหลบซ่อน เช่น วัชพืช
2. ดักจับหรือขุดบ่อดักข้างคันนา
3. ระบายน้ำหลังปักดำ หรือเมื่อข้าวเริ่มตั้งตัว

## จุดที่ต้องระวัง

- ใช้ตาข่ายกั้นทางน้ำร่วมกับการจัดการหอย
- ระดับน้ำเป็นเงื่อนไขสำคัญของประสิทธิภาพ
- ต้นฉบับระบุใช้สารกำจัดหอยเพียงครั้งเดียวต่อฤดู
- สูตร GB, GR และ WP ใช้ต่างกัน
- ห้ามนำอัตราสารชนิดหนึ่งไปใช้กับอีกชนิด
- การจัดการนกและปูนาเน้นวิธีเขตกรรมและกายภาพ

## แหล่งอ้างอิง
