# คำแนะนำการจัดการหนูในแปลงข้าวโพด

## หนูเป้าหมาย

ต้นฉบับระบุหนูหริ่งหางสั้น หนูหริ่งหางยาว หนูท้องขาวบ้าน หนูนาใหญ่ หนูพุกใหญ่ และหนูพุกเล็ก รวมถึงหนูสกุล _Bandicota_ และ _Rattus_

## ซิงค์ฟอสไฟด์ชนิดออกฤทธิ์เร็ว

| รายการ           | ข้อมูล                              |
| ---------------- | ----------------------------------- |
| ชื่อสามัญ        | ซิงค์ฟอสไฟด์                        |
| สูตร             | 80% powder                          |
| กลุ่ม            | 24A ตามต้นฉบับ                      |
| ระดับพิษและ LD50 | ร้ายแรง (12)                        |
| อัตรา            | สาร 10 กรัม ผสมเหยื่อล่อ 1 กิโลกรัม |
| เหยื่อตัวอย่าง   | ปลายข้าว ข้าวกล้อง หรือข้าวโพดป่น   |

### วิธีใช้

- ใช้จุดละประมาณ 1 ช้อนชา
- ใช้แกลบใหม่รองและกลบเหยื่อ
- วางเมื่อพบร่องรอยความเสียหาย
- วางทั่วแปลง ห่างกัน 5-10 เมตร ตามระดับประชากรหนู
- เหมาะกับระยะเตรียมแปลง
- ไม่ควรใช้เกิน 1 ครั้งต่อฤดู เพราะทำให้หนูเข็ดขยาดเหยื่อได้ง่าย

### วิธีวาง

- ใส่ภาชนะใส่เหยื่อ
- วางรอบแปลง โดยเฉพาะพื้นที่ติดดงหญ้าหรือชายป่า
- จุดละ 3-5 ก้อน ตามรอยทางเดินและรอบบริเวณเสียหาย
- ห้ามบริโภคหนูในบริเวณที่ใช้สารชนิดออกฤทธิ์ช้า

อัตรา “20 ก้อนต่อจุด จำนวน 10 จุดต่อไร่” กับข้อความ “จุดละ 3-5 ก้อน” ปรากฏร่วมกันในต้นฉบับและอาจไม่สอดคล้องกัน ต้องยึดฉลากผลิตภัณฑ์จริงเป็นหลัก

## เหยื่อโปรโตซัว

| รายการ      | ข้อมูล                                                |
| ----------- | ----------------------------------------------------- |
| ชนิด        | _Sarcocystis singaporensis_                           |
| ความเข้มข้น | 2 × 10^5 sporocysts                                   |
| อัตรา       | 20-25 ก้อน/ไร่                                        |
| วิธีวาง     | ใน bait station จุดละ 1-3 ก้อน ตามทางวิ่งหรือรอยทำลาย |
| ช่วงใช้     | ตั้งแต่หยอดเมล็ดจนถึงก่อนเก็บเกี่ยว                   |
| ระยะห่าง    | 15-20 วัน ตามประชากรหนู                               |
| ข้อควรระวัง | ป้องกันน้ำและแสงแดด และควรถูกกินภายใน 1 สัปดาห์       |

## จุดที่ต้องระวังอย่างยิ่ง

- สารหลายชนิดมีความเป็นพิษสูงมาก
- ต้องลดการเข้าถึงของเด็ก ปศุสัตว์ สัตว์เลี้ยง และสัตว์ป่า
- เก็บซากหนูและเหยื่อคงเหลือตามฉลาก
- ใช้เฉพาะผลิตภัณฑ์สำเร็จรูปที่ขึ้นทะเบียน
- ไม่ดัดแปลงเหยื่อหรือเพิ่มอัตราเอง
