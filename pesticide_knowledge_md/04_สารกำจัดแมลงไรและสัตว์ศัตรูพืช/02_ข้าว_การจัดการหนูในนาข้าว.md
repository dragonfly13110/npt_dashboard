# คำแนะนำการจัดการหนูในนาข้าว

## หนูเป้าหมายที่ต้นฉบับระบุ

หนูพุกใหญ่ หนูพุกเล็ก หนูนาใหญ่ หนูนาเล็ก หนูท้องขาวบ้าน หนูหริ่งนาหางสั้น และหนูหริ่งนาหางยาว รวมถึงการใช้เหยื่อโปรโตซัวกับหนูสกุล _Bandicota_ และ _Rattus_

## ซิงค์ฟอสไฟด์ ออกฤทธิ์เร็ว

| รายการ           | ข้อมูล                                                                |
| ---------------- | --------------------------------------------------------------------- |
| ชื่อสามัญ        | ซิงค์ฟอสไฟด์ (zinc phosphide)                                         |
| สูตร             | 80% powder                                                            |
| กลุ่ม            | 24A ตามต้นฉบับ                                                        |
| ระดับพิษและ LD50 | ร้ายแรง (12)                                                          |
| อัตรา            | สาร 10 กรัม ผสมเหยื่อล่อ 1 กิโลกรัม                                   |
| เหยื่อตัวอย่าง   | ปลายข้าว ข้าวกล้อง หรือข้าวโพดป่น                                     |
| วิธีวาง          | ก่อนปลูก วางตามคันนาและแหล่งอาศัย จุดละประมาณ 1 ช้อนชา ห่าง 5-10 เมตร |
| ข้อจำกัด         | ไม่ควรใช้เกิน 1 ครั้งต่อฤดูปลูก เพราะหนูเข็ดขยาดเหยื่อได้ง่าย         |

ต้นฉบับระบุว่ามีเหยื่อพิษสำเร็จรูปบรรจุซองประมาณ 10 กรัมด้วย ต้องใช้เฉพาะผลิตภัณฑ์ที่ขึ้นทะเบียนและตามฉลาก

| ---------------------------- | --------------------- | ----------------- | ------------------------------- |
| โฟลคูมาเฟน (flocoumafen)     | 0.005% wax block bait | ร้ายแรงมาก (0.56) | 20 ก้อน/จุด จำนวน 10 จุด/ไร่    |
| โบรมาดิโอโลน (bromadiolone)  | 0.005% wax block bait | ร้ายแรงมาก (0.56) | 20 ก้อน/จุด จำนวน 10 จุด/ไร่    |
| โบรดิฟาคูม (brodifacoum)     | 0.005% wax block bait | ร้ายแรงมาก (0.3)  | 20 ก้อน/จุด จำนวน 10 จุด/ไร่    |

### วิธีใช้ที่ต้นฉบับระบุ

- วางในภาชนะใส่เหยื่อบนทางเดินหนู ตามคันนา หรือใส่ในรูหนู
- วางตามแหล่งที่มีการระบาด
- การใช้โฟลคูมาเฟนระบุให้ใช้ 2-3 ครั้ง โดยครั้งถัดไปห่างจากครั้งแรก 30 และ 60 วัน
- วางแนวป้องกันรอบแปลงเพื่อลดการเคลื่อนย้ายเข้ามา
- ห้ามบริโภคหนูในบริเวณที่มีการใช้สารกำจัดหนูประเภทนี้

อัตราบางรายการสูงและมีความเป็นพิษมาก ต้องยึดฉลากผลิตภัณฑ์จริง ไม่คำนวณหรือดัดแปลงเหยื่อเองนอกคำแนะนำทางการ

## เหยื่อโปรโตซัว

| รายการ              | ข้อมูล                                             |
| ------------------- | -------------------------------------------------- |
| ชนิด                | _Sarcocystis singaporensis_                        |
| ความเข้มข้น         | 2 × 10^5 sporocysts                                |
| อัตรา               | 20-25 ก้อน/ไร่                                     |
| วิธีวาง             | จุดละ 1-3 ก้อน บริเวณทางวิ่งหรือรอยทำลายทั่วแปลง   |
| ช่วงใช้             | ตั้งแต่เริ่มปลูกถึงก่อนเกี่ยว                      |
| ความถี่             | ห่างกัน 15-20 วัน ตามระดับประชากรหนู               |
| การเก็บรักษาหลังวาง | ป้องกันน้ำและแสงแดด และควรถูกหนูกินภายใน 1 สัปดาห์ |

## จุดที่ต้องระวังอย่างยิ่ง

- สารหลายรายการมีค่า LD50 ต่ำมาก
- ต้องใช้ภาชนะใส่เหยื่อและลดการเข้าถึงของคน เด็ก ปศุสัตว์ และสัตว์ที่ไม่ใช่เป้าหมายตามฉลาก
- ห้ามใช้ภาชนะอาหารตวงหรือผสม
- ห้ามบริโภคหนูจากพื้นที่ที่ใช้สารออกฤทธิ์ช้าตามคำเตือนต้นฉบับ
- ซิงค์ฟอสไฟด์ไม่ควรใช้เกินหนึ่งครั้งต่อฤดู
- เก็บซากและเหยื่อคงเหลือตามข้อกำหนดฉลาก

## แหล่งอ้างอิง

ศรีจำนรรจ์ ศรีจันทรา และพฤทธิชาติ ปุญวัฒโท. 2568. _คำแนะนำการใช้สารป้องกันกำจัดศัตรูพืชจากงานวิจัย ปี 2568_. กรมวิชาการเกษตร. หน้า 69-70.
