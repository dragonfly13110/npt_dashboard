# เพลี้ยไฟพริกในส้มเปลือกล่อนภาคเหนือ: สถานการณ์ความต้านทานต่อสารกำจัดแมลง

> ข้อมูลนี้เป็นผลจากแหล่งปลูกในจังหวัดเชียงใหม่และสุโขทัยช่วงปี พ.ศ. 2565-2566 ต้องตรวจสถานการณ์ปัจจุบันของสวนและพื้นที่ก่อนนำไปวางแผน

## 1. เพลี้ยไฟพริกในส้มเปลือกล่อนภาคเหนือ

ศัตรูเป้าหมายคือเพลี้ยไฟพริก _Scirtothrips dorsalis_

### เกณฑ์การตายที่ต้นฉบับใช้ในแผนภาพ

| ระดับการตาย | เกณฑ์ตามต้นฉบับ                                        | ความหมายเชิงเฝ้าระวัง     |
| ----------- | ------------------------------------------------------ | ------------------------- |
| สูง         | อัตราแนะนำตาย 60-100% หรือสองเท่าอัตราแนะนำตาย 80-100% | แนวโน้มความต้านทานต่ำกว่า |
| ปานกลาง     | อัตราแนะนำตาย 21-59% หรือสองเท่าอัตราแนะนำตาย 41-79%   | แนวโน้มความต้านทานปานกลาง |
| ต่ำ         | อัตราแนะนำตาย 0-20% หรือสองเท่าอัตราแนะนำตาย 0-40%     | แนวโน้มความต้านทานสูงกว่า |

### คำแนะนำรายพื้นที่ตามตารางต้นฉบับ

| จังหวัด   | อำเภอ       | ไม่พบ/ความต้านทานต่ำ: ตัวเลือกตามต้นฉบับ                                                                                                                                                  | ความต้านทานปานกลาง                                                   | ความต้านทานสูง: ไม่ควรใช้ในการหมุนเวียนตามต้นฉบับ |
| --------- | ----------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------- | ------------------------------------------------- |
| เชียงใหม่ | ฝาง         | ฟิโพรนิล (2B), อิมิดาโคลพริด (4A), สไปนีโทแรม (5), อีมาเมกตินเบนโซเอต (6), คลอร์ฟีนาเพอร์ (13)                                                                                            | อะบาเมกติน (6), ไซแอนทรานิลิโพรล (28)                                | อะซีทามิพริด (4A)                                 |
| เชียงใหม่ | แม่อาย      | สไปนีโทแรม (5), อีมาเมกตินเบนโซเอต (6), คลอร์ฟีนาเพอร์ (13), ไซแอนทรานิลิโพรล (28)                                                                                                        | ฟิโพรนิล (2B), อิมิดาโคลพริด (4A), อะซีทามิพริด (4A), อะบาเมกติน (6) | ไม่ระบุรายการ                                     |
| สุโขทัย   | ศรีสัชนาลัย | ต้นฉบับระบุว่าไม่พบความต้านทานต่อฟิโพรนิล (2B), อิมิดาโคลพริด (4A), อะซีทามิพริด (4A), สไปนีโทแรม (5), อะบาเมกติน (6), อีมาเมกตินเบนโซเอต (6), คลอร์ฟีนาเพอร์ (13), ไซแอนทรานิลิโพรล (28) | ไม่ระบุรายการ                                                        | ไม่ระบุรายการ                                     |

### ประเด็นที่เห็นจากข้อมูล

- อะซีทามิพริดกลุ่ม 4A อยู่ระดับสูงที่อำเภอฝาง แต่ต้นฉบับระบุไม่พบความต้านทานที่ศรีสัชนาลัย แสดงให้เห็นว่าห้ามเหมารวมข้ามพื้นที่
- ที่แม่อาย สารกลุ่ม 4A สองรายการ ได้แก่ อิมิดาโคลพริดและอะซีทามิพริด อยู่ระดับปานกลาง จึงไม่ควรสลับกันโดยคิดว่าเป็นคนละกลุ่ม
- อะบาเมกตินและอีมาเมกตินเบนโซเอตอยู่กลุ่ม 6 เหมือนกัน แต่ระดับความต้านทานในพื้นที่เดียวกันอาจต่างกัน การวางแผนต้องพิจารณาทั้งกลุ่มและสารออกฤทธิ์
- สวนที่ยังได้ผลดีควรใช้โอกาสนี้ลดแรงคัดเลือก โดยสำรวจแมลง ใช้สารเมื่อถึงเกณฑ์ และเปลี่ยนกลุ่มตามรอบ

## จุดที่ต้องระวังและตรวจสอบ

- คำว่า “ไม่พบความต้านทาน” หมายถึงผลในประชากรและวิธีทดสอบช่วงนั้น ไม่ได้ยืนยันว่าสารจะควบคุมได้ดีตลอดไป
- ส่วนนี้ไม่มีสูตรผลิตภัณฑ์ อัตราใช้ วิธีพ่น หรือ PHI ห้ามนำชื่อสารไปใช้จากตารางนี้เพียงอย่างเดียว
- พืชตระกูลส้มเก็บผลเพื่อบริโภคและจำหน่าย ต้องตรวจฉลาก ระยะเว้นก่อนเก็บเกี่ยว และข้อกำหนดตลาดทุกครั้ง
- ดูคำแนะนำรายพืชที่ [พืชตระกูลส้ม: แมลงและไรศัตรูทั้งหมด](../04_สารกำจัดแมลงไรและสัตว์ศัตรูพืช/52_พืชตระกูลส้ม_แมลงและไรศัตรูทั้งหมด.md)
- ดูหลักการหมุนเวียนที่ [หลักการอ่านรหัสกลุ่มและหมุนเวียนสาร](../02_กลไกการออกฤทธิ์/01_หลักการอ่านรหัสกลุ่มและหมุนเวียนสาร.md)
