# หนอนกระทู้หอมในหอมแดง: สถานการณ์ความต้านทานต่อสารกำจัดแมลง

> ข้อมูลนี้มาจากแหล่งปลูกจังหวัดกาญจนบุรี สุพรรณบุรี นครปฐม และราชบุรี ช่วงปี พ.ศ. 2565-2566 ต้องตรวจสถานการณ์ปัจจุบันก่อนนำไปใช้

## 1. หนอนกระทู้หอมในหอมแดง

ศัตรูเป้าหมายคือหนอนกระทู้หอม _Spodoptera exigua_

### เกณฑ์ค่า RF ในแผนภาพต้นฉบับ

| ช่วงค่า RF      | ระดับในแผนภาพ              |
| --------------- | -------------------------- |
| RF ≤ 1 เท่า     | ไม่เพิ่มจากประชากรอ้างอิง  |
| RF >1-10 เท่า   | เพิ่มเล็กน้อยถึงระดับหนึ่ง |
| RF >10-20 เท่า  | เพิ่มชัดเจน                |
| RF >20-50 เท่า  | สูง                        |
| RF >50-100 เท่า | สูงมาก                     |
| RF >100 เท่า    | สูงรุนแรง                  |

ตารางต่อไปนี้ถอดตามคำแนะนำของต้นฉบับที่จัดสารเป็นความต้านทานต่ำ ปานกลาง และสูง

| จังหวัด    | อำเภอ       | ความต้านทานต่ำ: ใช้เป็นตัวเลือกหมุนเวียนตามต้นฉบับ | ความต้านทานปานกลาง                       | ความต้านทานสูง: ไม่ควรใช้ในการหมุนเวียนตามต้นฉบับ                   |
| ---------- | ----------- | -------------------------------------------------- | ---------------------------------------- | ------------------------------------------------------------------- |
| สุพรรณบุรี | ศรีประจันต์ | สไปนีโทแรม (5)                                     | คลอร์ฟีนาเพอร์ (13), อินดอกซาคาร์บ (22A) | อีมาเมกตินเบนโซเอต (6), คลอแรนทรานิลิโพรล (28)                      |
| นครปฐม     | บางเลน      | สไปนีโทแรม (5)                                     | คลอร์ฟีนาเพอร์ (13), อินดอกซาคาร์บ (22A) | อีมาเมกตินเบนโซเอต (6), คลอแรนทรานิลิโพรล (28)                      |
| กาญจนบุรี  | ท่าม่วง     | สไปนีโทแรม (5)                                     | คลอร์ฟีนาเพอร์ (13), อินดอกซาคาร์บ (22A) | อีมาเมกตินเบนโซเอต (6), คลอแรนทรานิลิโพรล (28)                      |
| ราชบุรี    | จอมบึง      | สไปนีโทแรม (5)                                     | อินดอกซาคาร์บ (22A)                      | อีมาเมกตินเบนโซเอต (6), คลอร์ฟีนาเพอร์ (13), คลอแรนทรานิลิโพรล (28) |

### หลักการนำไปใช้

- ผลจากทั้ง 4 พื้นที่ตรงกันว่า สไปนีโทแรมกลุ่ม 5 อยู่ในระดับความต้านทานต่ำตามข้อมูลช่วงที่ศึกษา
- อีมาเมกตินเบนโซเอตกลุ่ม 6 และคลอแรนทรานิลิโพรลกลุ่ม 28 อยู่ในระดับความต้านทานสูงทุกพื้นที่ในตาราง
- คลอร์ฟีนาเพอร์กลุ่ม 13 อยู่ระดับปานกลางในสามพื้นที่ แต่สูงที่จอมบึง แสดงให้เห็นว่าความต้านทานต้องพิจารณารายพื้นที่
- แม้สารหนึ่งยังอยู่ระดับต่ำ ไม่ควรใช้กลุ่มเดิมต่อเนื่อง เพราะจะเร่งการคัดเลือกประชากรต้านทาน

## จุดที่ต้องระวังและตรวจสอบ

- ค่า RF เป็นอัตราส่วนความต้านทานจากการทดสอบ ไม่ใช่อัตราใช้ในแปลง
- ส่วนนี้ไม่มีสูตรผลิตภัณฑ์ อัตราใช้ หรือ PHI ห้ามนำชื่อสารไปพ่นโดยอ้างตารางนี้เพียงอย่างเดียว
- ข้อมูลปี 2565-2566 อาจเปลี่ยนแล้ว โดยเฉพาะพื้นที่ปลูกหอมที่ใช้สารเข้มข้นต่อเนื่อง
- ดูคำแนะนำรายพืชที่ [หอมแดง หอมแบ่ง หอมหัวใหญ่ และกระเทียม](../04_สารกำจัดแมลงไรและสัตว์ศัตรูพืช/63_หอมแดงหอมแบ่งหอมหัวใหญ่และกระเทียม_แมลงและไรศัตรู.md)
- ดูหลักการหมุนเวียนที่ [หลักการอ่านรหัสกลุ่มและหมุนเวียนสาร](../02_กลไกการออกฤทธิ์/01_หลักการอ่านรหัสกลุ่มและหมุนเวียนสาร.md)
