# หนอนกระทู้ข้าวโพดลายจุดในข้าวโพด: สถานการณ์ความต้านทานต่อสารกำจัดแมลง

> ข้อมูลนี้มาจากการสำรวจประชากรหนอนกระทู้ข้าวโพดลายจุดในพื้นที่ต่าง ๆ ของประเทศไทยช่วงปี พ.ศ. 2562-2566 และคำแนะนำรายอำเภอในต้นฉบับ ข้อมูลความต้านทานเปลี่ยนแปลงได้ตามพื้นที่และประวัติการใช้สาร จึงต้องสำรวจและตรวจข้อมูลปัจจุบันก่อนใช้วางแผน

## 1. หนอนกระทู้ข้าวโพดลายจุดในข้าวโพด

ศัตรูเป้าหมายคือหนอนกระทู้ข้าวโพดลายจุด _Spodoptera frugiperda_

### สารที่ต้นฉบับแสดงในแผนที่ความต้านทาน

ต้นฉบับแสดงแผนที่แยกตามสาร 6 รายการ ได้แก่

| ชื่อสาร                                 | กลุ่มกลไกการออกฤทธิ์ |
| --------------------------------------- | -------------------: |
| สไปนีโทแรม (spinetoram)                 |                    5 |
| อีมาเมกตินเบนโซเอต (emamectin benzoate) |                    6 |
| คลอร์ฟีนาเพอร์ (chlorfenapyr)           |                   13 |
| ลูเฟนนูรอน (lufenuron)                  |                   15 |
| อินดอกซาคาร์บ (indoxacarb)              |                  22A |
| คลอแรนทรานิลิโพรล (chlorantraniliprole) |                   28 |

### เกณฑ์ค่า RC ในแผนที่ต้นฉบับ

| ช่วงค่า RC       | สัญลักษณ์ระดับในแผนที่                            |
| ---------------- | ------------------------------------------------- |
| RC ≤ 1 เท่า      | ประชากรทดสอบไม่สูงกว่าประชากรอ้างอิงตามเกณฑ์ในภาพ |
| RC > 1.1-2 เท่า  | เพิ่มขึ้นเล็กน้อย                                 |
| RC > 2.1-5 เท่า  | เพิ่มขึ้นระดับหนึ่ง                               |
| RC > 5.1-10 เท่า | เพิ่มขึ้นมาก                                      |
| RC > 10 เท่า     | เพิ่มขึ้นสูง                                      |

> ค่า RC เป็นค่าที่ใช้เปรียบเทียบความไวของประชากรแมลงในการทดลอง ไม่ใช่อัตราใช้สารในแปลง

### หลักการนำไปใช้

- ตารางแสดงผลเฉพาะประชากรแมลงและช่วงเวลาที่ศึกษา ไม่ได้หมายความว่าสารที่อยู่ระดับต่ำจะใช้ซ้ำต่อเนื่องได้
- อู่ทองและท่าม่วงมีอีมาเมกตินเบนโซเอตกลุ่ม 6 อยู่ระดับปานกลาง ขณะที่พื้นที่อื่นในตารางอยู่ระดับต่ำ แสดงให้เห็นความแตกต่างรายพื้นที่
- ศรีประจันต์มีลูเฟนนูรอนกลุ่ม 15 อยู่ระดับปานกลาง แต่พื้นที่อื่นในตารางส่วนใหญ่อยู่ระดับต่ำ
- การหมุนเวียนต้องเปลี่ยน **กลุ่มกลไกการออกฤทธิ์** ไม่ใช่เปลี่ยนเพียงชื่อการค้า
- หลังพ่นควรสุ่มตรวจหนอนรอดและบันทึกประวัติกลุ่มสาร เพื่อจับสัญญาณประสิทธิภาพลดลงก่อนเกิดปัญหารุนแรง

## จุดที่ต้องระวังและตรวจสอบ

- แผนที่ต้นฉบับมีจุดสำรวจมากกว่าพื้นที่ในตารางคำแนะนำ ไฟล์นี้ยึดรายชื่อและระดับจากตารางหน้าพิมพ์ 222-223 เป็นหลัก
- ค่า RC ไม่ใช่จำนวนครั้งพ่น ความเข้มข้น อัตราต่อไร่ หรือค่า PHI
- ส่วนสถานการณ์ความต้านทานไม่ระบุสูตรผลิตภัณฑ์ อัตราใช้ เกณฑ์เริ่มพ่น และระยะเว้นก่อนเก็บเกี่ยว ห้ามนำชื่อสารจากตารางไปใช้โดยลำพัง
- ดูอัตราและวิธีใช้จาก [ข้าวโพด: หนอนกระทู้ข้าวโพดลายจุด](../04_สารกำจัดแมลงไรและสัตว์ศัตรูพืช/06_ข้าวโพด_หนอนกระทู้ข้าวโพดลายจุด.md)
- ดูหลักการหมุนเวียนที่ [หลักการอ่านรหัสกลุ่มและหมุนเวียนสาร](../02_กลไกการออกฤทธิ์/01_หลักการอ่านรหัสกลุ่มและหมุนเวียนสาร.md)
