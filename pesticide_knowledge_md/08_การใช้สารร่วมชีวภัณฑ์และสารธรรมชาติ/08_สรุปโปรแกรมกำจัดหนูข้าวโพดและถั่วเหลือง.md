# สรุปโปรแกรมสารกำจัดหนูร่วมกับเหยื่อโปรโตซัวในข้าวโพดเลี้ยงสัตว์และถั่วเหลือง

> **คำเตือนระดับสูง: สารกำจัดหนูและเหยื่อพิษอาจทำให้คน เด็ก สัตว์เลี้ยง สัตว์ป่า และสัตว์ที่ไม่ใช่เป้าหมายเสียชีวิต ต้องใช้เฉพาะผลิตภัณฑ์ที่ได้รับอนุญาต วางในภาชนะหรือสถานีเหยื่อที่ป้องกันการเข้าถึง เก็บกู้เหยื่อและซากสัตว์อย่างปลอดภัย และปฏิบัติตามฉลากอย่างเคร่งครัด**

## 1. โปรแกรมข้าวโพดเลี้ยงสัตว์

| ระยะ/อายุหลังงอก         | โปรแกรมตามต้นฉบับ                                                                                              |
| ------------------------ | -------------------------------------------------------------------------------------------------------------- |
| ระยะเตรียมดิน หรือ 0 วัน | zinc phosphide 0.8-1% จุดละ 1 ช้อนชา ระยะห่าง 5-10 เมตร ให้ครอบคลุมแปลงและรอบแปลง                              |
| 20 วัน                   | ไม่ระบุการใช้                                                                                                  |
| 50 วัน                   | ไม่ระบุการใช้                                                                                                  |
| 70 วัน                   | flocoumafen 0.005% จุดละ 20 ก้อน ร่วมกับ _S. singaporensis_ จุดละ 3 ก้อน อย่างละ 10 จุด/ไร่ ระยะห่าง 5-10 เมตร |
| 90 วัน                   | ใช้ flocoumafen และเหยื่อโปรโตซัวในอัตราเดิม                                                                   |
| เก็บเกี่ยว               | ประมาณ 120 วันหลังงอกตามตาราง                                                                                  |
| PHI                      | ไม่ระบุ ต้องตรวจฉลากและข้อกำหนดปัจจุบัน                                                                        |

| 20 วัน                   | flocoumafen 0.005% จุดละ 20 ก้อน ร่วมกับ _S. singaporensis_ จุดละ 3 ก้อน อย่างละ 10 จุด/ไร่ ระยะห่าง 5-10 เมตร |
| 50 วัน                   | ใช้ flocoumafen และเหยื่อโปรโตซัวในอัตราเดิม                                                                   |

## 3. ตารางเปรียบเทียบและหลักการใช้

| ประเด็น                    | ข้าวโพดเลี้ยงสัตว์             | ถั่วเหลือง                     |
| -------------------------- | ------------------------------ | ------------------------------ |
| ครั้งที่ 1                 | เตรียมดิน: zinc phosphide      | เตรียมดิน: zinc phosphide      |
| ครั้งที่ 2                 | 70 วัน: flocoumafen + โปรโตซัว | 20 วัน: flocoumafen + โปรโตซัว |
| ครั้งที่ 3                 | 90 วัน: flocoumafen + โปรโตซัว | 50 วัน: flocoumafen + โปรโตซัว |
| จำนวนจุดช่วงสารออกฤทธิ์ช้า | อย่างละ 10 จุด/ไร่             | อย่างละ 10 จุด/ไร่             |
| ระยะห่างจุด                | 5-10 เมตร                      | 5-10 เมตร                      |
| เก็บเกี่ยวตามตาราง         | 120 วัน                        | 70 วัน                         |

### หลักการสำคัญ

- โปรแกรมต่างกันที่ช่วงอายุพืช ห้ามนำวันของข้าวโพดไปใช้กับถั่วเหลืองหรือกลับกัน
- ระยะการใช้ควรสัมพันธ์กับการสำรวจร่องรอยและประชากรหนู ไม่ควรวางเหยื่อโดยไม่ตรวจพื้นที่
- การเติมเหยื่อและเก็บกู้ต้องทำตามฉลากและระบบสถานีเหยื่อที่ปลอดภัย

## จุดที่ต้องระวังและตรวจสอบ

- คำเตือนระดับสูง: สารกำจัดหนูและเหยื่อพิษอาจทำให้คน เด็ก สัตว์เลี้ยง สัตว์ป่า และสัตว์ที่ไม่ใช่เป้าหมายเสียชีวิต ต้องใช้เฉพาะผลิตภัณฑ์ที่ได้รับอนุญาต วางในภาชนะหรือสถานีเหยื่อที่ป้องกันการเข้าถึง เก็บกู้เหยื่อและซากสัตว์อย่างปลอดภัย และปฏิบัติตามฉลากอย่างเคร่งครัด
- ตารางเป็นบทสรุปจากงานวิจัยเฉพาะระบบการผลิต ไม่ใช้แทนทะเบียน ฉลาก หรือการประเมินหนูระดับแปลง
- ระยะเก็บเกี่ยว 120 และ 70 วันเป็นค่าจากตารางต้นฉบับ อาจไม่ตรงกับพันธุ์หรือฤดูปลูกจริง
- ห้ามนำอัตราของ zinc phosphide หรือ flocoumafen ไปใช้กับสูตรและความเข้มข้นอื่น
- ต้องป้องกันพิษต่อสัตว์ที่ไม่ใช่เป้าหมาย เก็บกู้เหยื่อเหลือและซากหนู และบันทึกตำแหน่งสถานีเหยื่อทุกจุด
- ต้นฉบับไม่ระบุ PHI ต้องตรวจฉลากและข้อกำหนดปัจจุบัน
- ข้อมูลนี้ใช้เพื่อการเรียนรู้และค้นข้อมูล ไม่ใช้แทนทะเบียน ฉลากผลิตภัณฑ์ หรือข้อกำหนดกฎหมายปัจจุบัน ก่อนใช้จริงต้องตรวจสอบผลิตภัณฑ์ พืชเป้าหมาย สูตร อัตรา วิธีใช้ ลำดับการใช้ ความเข้ากันได้ ระยะเว้นก่อนเก็บเกี่ยว และข้อจำกัดบนฉลากทุกครั้ง
