# ผักกวางตุ้ง: โปรแกรมสารกำจัดแมลงร่วมกับไส้เดือนฝอยควบคุมด้วงหมัดผัก

ต้นฉบับกำหนดระยะเก็บเกี่ยวผลผลิตประมาณ **35-40 วันหลังหว่าน** และเสนอ 2 รูปแบบการใช้สารกำจัดแมลงร่วมกับไส้เดือนฝอย _Steinernema carpocapsae_

| ------------- | ------------------------------- | ----------------------------- | -------------------------------------------- | -------------------- |
| 0 วัน         | ไม่ใช้                          | -                             | -                                            | -                    |
| 5 วัน         | ไม่ใช้                          | -                             | -                                            | -                    |

### การหมุนเวียนกลุ่มสาร

ลำดับกลุ่มในรูปแบบนี้คือ **21A → 2B → 2B → 4A → ชีวภัณฑ์** โดยฟิโพรนิลถูกใช้ติดต่อกัน 2 ครั้งตามต้นฉบับ ต้องไม่ขยายจำนวนครั้งหรือช่วงใช้เอง

### การหมุนเวียนกลุ่มสาร

ลำดับของสารเคมีในรูปแบบนี้คือ **21A → 21A → 2B** สลับช่วงต้นและช่วงท้ายด้วยไส้เดือนฝอยตามตารางต้นฉบับ การใช้ไส้เดือนฝอยไม่ถือเป็นการพ่นสารเคมีอีกกลุ่มหนึ่ง และต้องจัดการคุณภาพชีวภัณฑ์แยกต่างหาก

## จุดที่ต้องระวังและตรวจสอบ

- อะซีทามิพริดสูตร 20% SP เป็นต้นฉบับระบุอัตรา **50 มล./น้ำ 20 ลิตร** ซึ่งผิดสังเกตสำหรับสูตรผง ต้องเก็บข้อมูลเดิมพร้อมธงตรวจสอบ และห้ามใช้จนกว่าจะยืนยันหน่วยจากฉลากหรืองานวิจัยต้นทาง
- ต้นฉบับไม่ระบุความเข้มข้นหรือสูตรของโทลเฟนไพแรด ต้องตรวจผลิตภัณฑ์จริงก่อนใช้
- ห้ามนำตารางนี้ไปใช้กับผักชนิดอื่น หรือเปลี่ยนวันพ่นโดยไม่ตรวจอายุเก็บเกี่ยวและ PHI ของสารแต่ละตัว
- ไส้เดือนฝอยเป็นสิ่งมีชีวิต ต้องตรวจวันหมดอายุ วิธีเก็บรักษา คุณภาพน้ำ อุณหภูมิ แสงแดด และวิธีราดจากฉลากผลิตภัณฑ์
- ไม่ผสมไส้เดือนฝอยกับสารกำจัดแมลงในถังเดียวกัน เว้นแต่มีข้อมูลความเข้ากันได้ที่ยืนยันชัดเจน
- ข้อมูลนี้ใช้เพื่อการเรียนรู้และค้นข้อมูล ไม่ใช้แทนทะเบียน ฉลากผลิตภัณฑ์ หรือข้อกำหนดกฎหมายปัจจุบัน ก่อนใช้จริงต้องตรวจสอบผลิตภัณฑ์ พืชเป้าหมาย สูตร อัตรา วิธีใช้ ความเข้ากันได้ ระยะการใช้ ระยะเว้นก่อนเก็บเกี่ยว และข้อจำกัดบนฉลากทุกครั้ง
