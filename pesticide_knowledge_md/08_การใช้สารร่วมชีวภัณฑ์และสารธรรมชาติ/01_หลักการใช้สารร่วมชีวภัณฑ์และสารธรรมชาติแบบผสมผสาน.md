# หลักการใช้สารป้องกันกำจัดศัตรูพืชร่วมกับชีวภัณฑ์และสารธรรมชาติแบบผสมผสาน

## 1. เหตุผลที่ต้องใช้หลายวิธีร่วมกัน

ต้นฉบับอธิบายว่า ศัตรูพืชทำให้ผลผลิตเสียหายอย่างมาก และสภาพอากาศร้อนชื้นของประเทศไทยเอื้อต่อการแพร่พันธุ์และการระบาดได้ตลอดปี จึงควรเลือกวิธีจัดการให้สอดคล้องกับระดับการระบาด ไม่พึ่งวิธีใดวิธีหนึ่งต่อเนื่อง

วิธีที่ไม่ใช้สารเคมีซึ่งต้นฉบับยกตัวอย่าง ได้แก่

- พันธุ์ต้านทาน
- การเขตกรรม
- วิธีกล
- วิธีทางกายภาพ
- ชีววิธี

วิธีเหล่านี้เหมาะสำหรับการป้องกันและการระบาดระดับต่ำ ส่วนกรณีระบาดรุนแรงอาจจำเป็นต้องใช้สารเคมีเพื่อลดประชากรหรือความรุนแรงลงมาให้อยู่ต่ำกว่าระดับที่ก่อความเสียหายทางเศรษฐกิจ

## 2. หลัก IPC และ IPM ตามต้นฉบับ

ต้นฉบับใช้แนวคิดการป้องกันกำจัดโดยวิธีผสมผสาน หรือการบริหารศัตรูพืช โดยมุ่งใช้วิธีมากกว่า 1 วิธีร่วมกัน ไม่กำหนดให้สารเคมีหรือชีวภัณฑ์เป็นคำตอบเดียวทุกสถานการณ์

| ระดับสถานการณ์               | แนวทางตามหลักการในต้นฉบับ                                                               |
| ---------------------------- | --------------------------------------------------------------------------------------- |
| ก่อนพบหรือพบศัตรูพืชระดับต่ำ | ใช้การป้องกัน การสำรวจ พันธุ์ต้านทาน เขตกรรม วิธีกล กายภาพ และชีววิธี                   |
| การระบาดเพิ่มขึ้น            | ประเมินระดับการทำลายและเสริมวิธีที่เหมาะสมหลายวิธี                                      |
| การระบาดรุนแรง               | ใช้สารเคมีเท่าที่จำเป็นเพื่อลดประชากรหรือความรุนแรง แล้วกลับเข้าสู่ระบบจัดการแบบผสมผสาน |

## 3. รูปแบบการใช้สารร่วมกับชีวภัณฑ์หรือสารธรรมชาติ

### กรณีแมลงศัตรูพืช

- หมุนเวียนสารกำจัดแมลงตามกลุ่มกลไกการออกฤทธิ์ เพื่อชะลอความต้านทาน
- ผสมผสานการใช้ชีวภัณฑ์ เพื่อเพิ่มประสิทธิภาพหรือช่วยลดการตกค้างตามโปรแกรมที่ผ่านการทดสอบ

### กรณีโรคพืช

- ใช้สารป้องกันกำจัดโรคพืชเพื่อลดความรุนแรงของโรค
- ตามด้วยชีวภัณฑ์หรือสารธรรมชาติก่อนเก็บเกี่ยว ตามโปรแกรมที่ผ่านการทดสอบ เพื่อช่วยลดการตกค้างในผลผลิต

> คำว่า “ร่วมกับ” ในบทนี้หมายถึงการจัดโปรแกรมแบบผสมผสาน ไม่ได้หมายความว่าสามารถผสมผลิตภัณฑ์ทุกชนิดลงถังเดียวกันได้

## 4. การนำผลวิจัยไปใช้ในแปลง

ต้นฉบับระบุว่าคำแนะนำผ่านการทดสอบประสิทธิภาพในสภาพแปลงและมีการวิเคราะห์พิษตกค้าง แต่ผู้ใช้ยังต้องปรับให้เหมาะกับสภาพแปลงจริงโดยไม่เปลี่ยนอัตรา สูตร หรือลำดับการใช้โดยไม่มีหลักฐาน

### รายการตรวจสอบก่อนนำไปใช้

1. ยืนยันชนิดพืช ศัตรูพืช และระดับการระบาด
2. ตรวจทะเบียนและฉลากปัจจุบันของสารและชีวภัณฑ์
3. แยกการพ่นสาร การราดชีวภัณฑ์ และการใช้สารธรรมชาติตามลำดับที่ต้นฉบับกำหนด
4. ตรวจความเข้ากันได้ก่อนผสมถัง และไม่ผสมเองเมื่อไม่มีข้อมูล
5. ตรวจ PHI สารตกค้าง และข้อกำหนดตลาดปลายทาง
6. บันทึกผลการควบคุมและผลกระทบต่อศัตรูธรรมชาติ

## จุดที่ต้องระวังและตรวจสอบ

- ตัวอย่างในบทต่อไปเป็นโปรแกรมจากงานวิจัยเฉพาะพืช ศัตรูพืช สูตร อัตรา และช่วงเวลา ห้ามนำไปใช้ข้ามพืชหรือข้ามศัตรูพืช
- ชีวภัณฑ์เป็นสิ่งมีชีวิตหรือมีสารออกฤทธิ์เฉพาะ การเก็บรักษา คุณภาพผลิตภัณฑ์ น้ำที่ใช้ และสภาพอากาศมีผลต่อประสิทธิภาพ
- การลดพิษตกค้างที่รายงานในงานวิจัยไม่รับประกันผลเท่ากันทุกแปลง และไม่ใช้แทนการตรวจสารตกค้างจริง
- ต้องตรวจ PHI ของสารเคมีทุกตัว แม้โปรแกรมจะตามด้วยชีวภัณฑ์หรือสารธรรมชาติ
- ข้อมูลนี้ใช้เพื่อการเรียนรู้และค้นข้อมูล ไม่ใช้แทนทะเบียน ฉลากผลิตภัณฑ์ หรือข้อกำหนดกฎหมายปัจจุบัน ก่อนใช้จริงต้องตรวจสอบผลิตภัณฑ์ พืชเป้าหมาย สูตร อัตรา วิธีใช้ ความเข้ากันได้ ระยะการใช้ ระยะเว้นก่อนเก็บเกี่ยว และข้อจำกัดบนฉลากทุกครั้ง
