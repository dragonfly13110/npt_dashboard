# ข้าวโพด: การใช้สารกำจัดหนูร่วมกับเหยื่อโปรโตซัว Sarcocystis singaporensis

> **คำเตือนระดับสูง: สารกำจัดหนูและเหยื่อพิษอาจทำให้คน เด็ก สัตว์เลี้ยง สัตว์ป่า และสัตว์ที่ไม่ใช่เป้าหมายเสียชีวิต ต้องใช้เฉพาะผลิตภัณฑ์ที่ได้รับอนุญาต วางในภาชนะหรือสถานีเหยื่อที่ป้องกันการเข้าถึง เก็บกู้เหยื่อและซากสัตว์อย่างปลอดภัย และปฏิบัติตามฉลากอย่างเคร่งครัด**

โปรแกรมต้นฉบับวางเหยื่อ 3 ช่วง ได้แก่ ระยะเตรียมดิน 70 วันหลังงอก และ 90 วันหลังงอก โดยใช้สารออกฤทธิ์เร็วเพียงครั้งเดียวในช่วงต้น จากนั้นใช้สารออกฤทธิ์ช้าร่วมกับเหยื่อโปรโตซัว

## 1. ระยะเตรียมดินถึงเริ่มหยอดเมล็ด

| รายการ               | รายละเอียดตามต้นฉบับ                                                         |
| -------------------- | ---------------------------------------------------------------------------- |
| สาร                  | zinc phosphide, Zn3P2 80% powder                                             |
| ลักษณะการออกฤทธิ์    | สารกำจัดหนูชนิดออกฤทธิ์เร็ว                                                  |
| ความเข้มข้นเหยื่อพิษ | 0.8-1%                                                                       |
| การเตรียมตามต้นฉบับ  | สาร 10 กรัม ผสมเหยื่อล่อ เช่น ปลายข้าวหรือข้าวโพดป่น 1 กิโลกรัม              |
| ปริมาณต่อจุด         | 1 ช้อนชา ประมาณ 5-10 กรัม                                                    |
| ระยะห่าง             | 5-10 เมตร ขึ้นกับประชากรหนู                                                  |
| ตำแหน่ง              | ให้ทั่วแปลงและบริเวณโดยรอบ ใช้ภาชนะใส่เหยื่อ หรือแกลบใหม่รองและกลบตามต้นฉบับ |
| ข้อจำกัด             | ใช้เพียง 1 ครั้งต่อฤดูปลูก                                                   |
| PHI                  | ไม่ระบุ ต้องตรวจฉลากและข้อกำหนดปัจจุบัน                                      |

ต้นฉบับระบุว่าช่วงนี้หนูยังมีอาหารธรรมชาติน้อย จึงเหมาะกับการลดประชากรอย่างรวดเร็ว แต่การใช้จริงต้องควบคุมการเข้าถึงของคนและสัตว์ที่ไม่ใช่เป้าหมายอย่างเข้มงวด

## 2. ระยะออกดอกติดฝัก 70 วันหลังงอก

| รายการ                                     | อัตราและวิธีตามต้นฉบับ                                                 |
| ------------------------------------------ | ---------------------------------------------------------------------- |
| flocoumafen 0.005%                         | 20 ก้อน/จุด จำนวน 10 จุด/ไร่ ในภาชนะใส่เหยื่อ                          |
| เหยื่อโปรโตซัว _Sarcocystis singaporensis_ | 3 ก้อน/จุด จำนวน 10 จุด/ไร่ ในภาชนะใส่เหยื่อ                           |
| ระยะห่าง                                   | 5-10 เมตร                                                              |
| ตำแหน่ง                                    | จุดพบร่องรอยหนู ทางวิ่ง รอบแปลงติดป่า คันนา หรือดงหญ้า ให้ครอบคลุมแปลง |
| PHI                                        | ไม่ระบุ ต้องตรวจฉลากและข้อกำหนดปัจจุบัน                                |

ต้นฉบับระบุว่าการใช้ร่วมกันเพิ่มประสิทธิภาพต่อหนูสกุลพุก (_Bandicota_) และหนูสกุลท้องขาว (_Rattus_) แต่ไม่ได้เพิ่มประสิทธิภาพต่อหนูหริ่ง (_Mus_) และมีต้นทุนสูงขึ้น

## 3. ระยะออกดอกติดฝัก 90 วันหลังงอกและต้นทุน

ใช้ flocoumafen 0.005% และเหยื่อโปรโตซัวในอัตราและตำแหน่งเดียวกับช่วง 70 วันหลังงอก พร้อมตรวจจำนวนเหยื่อที่ถูกกินและเติม flocoumafen ให้เท่ากับจำนวนเริ่มต้น ส่วนเหยื่อโปรโตซัวให้คงไว้ 3 ก้อน/จุดตามต้นฉบับ

| รายการต้นทุนตามต้นฉบับ |    บาท/ไร่ |
| ---------------------- | ---------: |
| Zinc phosphide         |      35.00 |
| Flocoumafen 0.005%     |     665.60 |
| เหยื่อโปรโตซัว         |     120.00 |
| **รวม**                | **820.00** |

### สรุปลำดับใช้

1. เตรียมดินถึงเริ่มหยอดเมล็ด: zinc phosphide ออกฤทธิ์เร็ว 1 ครั้ง
2. 70 วันหลังงอก: flocoumafen ออกฤทธิ์ช้าร่วมกับเหยื่อโปรโตซัว
3. 90 วันหลังงอก: flocoumafen ออกฤทธิ์ช้าร่วมกับเหยื่อโปรโตซัว

## จุดที่ต้องระวังและตรวจสอบ

- คำเตือนระดับสูง: สารกำจัดหนูและเหยื่อพิษอาจทำให้คน เด็ก สัตว์เลี้ยง สัตว์ป่า และสัตว์ที่ไม่ใช่เป้าหมายเสียชีวิต ต้องใช้เฉพาะผลิตภัณฑ์ที่ได้รับอนุญาต วางในภาชนะหรือสถานีเหยื่อที่ป้องกันการเข้าถึง เก็บกู้เหยื่อและซากสัตว์อย่างปลอดภัย และปฏิบัติตามฉลากอย่างเคร่งครัด
- Zinc phosphide และ flocoumafen เป็นสารพิษร้ายแรง ห้ามวางเหยื่อเปิดโล่งในจุดที่เด็ก คนงาน สัตว์เลี้ยง ปศุสัตว์ หรือสัตว์ป่าเข้าถึงได้
- ห้ามเพิ่มอัตรา จำนวนจุด หรือความเข้มข้นจากต้นฉบับ และห้ามนำสูตรนี้ไปใช้กับผลิตภัณฑ์ความเข้มข้นอื่น
- ต้องตรวจทะเบียน ฉลาก วิธีทำลายภาชนะ เหยื่อเหลือ และซากหนู รวมถึงมาตรการป้องกันพิษทุติยภูมิในปัจจุบัน
- เหยื่อโปรโตซัวมีผลแตกต่างตามสกุลหนู ต้นฉบับระบุว่าไม่ได้เพิ่มประสิทธิภาพต่อหนูหริ่ง (_Mus_)
- ต้นทุน 820 บาท/ไร่เป็นข้อมูล ณ เวลาทำวิจัย ไม่ใช่ราคาปัจจุบัน
- ต้นฉบับไม่ระบุ PHI ต้องตรวจฉลากและข้อกำหนดของผลผลิตก่อนใช้
- ข้อมูลนี้ใช้เพื่อการเรียนรู้และค้นข้อมูล ไม่ใช้แทนทะเบียน ฉลากผลิตภัณฑ์ หรือข้อกำหนดกฎหมายปัจจุบัน ก่อนใช้จริงต้องตรวจสอบผลิตภัณฑ์ พืชเป้าหมาย สูตร อัตรา วิธีใช้ ลำดับการใช้ ความเข้ากันได้ ระยะเว้นก่อนเก็บเกี่ยว และข้อจำกัดบนฉลากทุกครั้ง
