# ถั่วเหลือง: การใช้สารกำจัดหนูร่วมกับเหยื่อโปรโตซัว Sarcocystis singaporensis

> **คำเตือนระดับสูง: สารกำจัดหนูและเหยื่อพิษอาจทำให้คน เด็ก สัตว์เลี้ยง สัตว์ป่า และสัตว์ที่ไม่ใช่เป้าหมายเสียชีวิต ต้องใช้เฉพาะผลิตภัณฑ์ที่ได้รับอนุญาต วางในภาชนะหรือสถานีเหยื่อที่ป้องกันการเข้าถึง เก็บกู้เหยื่อและซากสัตว์อย่างปลอดภัย และปฏิบัติตามฉลากอย่างเคร่งครัด**

โปรแกรมต้นฉบับใช้สารกำจัดหนู 3 ครั้ง ได้แก่ ระยะเตรียมดิน 20 วันหลังงอก และ 50 วันหลังงอก โดยใช้สารออกฤทธิ์เร็วช่วงแรก และใช้สารออกฤทธิ์ช้าร่วมกับเหยื่อโปรโตซัวในสองช่วงหลัง

## 1. ระยะเตรียมดิน

| รายการ               | รายละเอียดตามต้นฉบับ                                                    |
| -------------------- | ----------------------------------------------------------------------- |
| สาร                  | zinc phosphide, Zn3P2 80% powder                                        |
| ลักษณะการออกฤทธิ์    | สารกำจัดหนูชนิดออกฤทธิ์เร็ว                                             |
| ความเข้มข้นเหยื่อพิษ | 0.8-1%                                                                  |
| การเตรียมตามต้นฉบับ  | สาร 10 กรัม ผสมปลายข้าวหรือข้าวโพดป่น 1 กิโลกรัม                        |
| ปริมาณต่อจุด         | 1 ช้อนชา ประมาณ 5-10 กรัม                                               |
| ระยะห่าง             | 5-10 เมตร ตามประชากรหนู                                                 |
| ตำแหน่ง              | ให้ทั่วแปลงและรอบแปลง ใช้ภาชนะใส่เหยื่อ หรือแกลบใหม่รองและกลบตามต้นฉบับ |
| ข้อจำกัด             | ใช้เพียง 1 ครั้งต่อฤดูปลูก                                              |
| PHI                  | ไม่ระบุ ต้องตรวจฉลากและข้อกำหนดปัจจุบัน                                 |

## 2. ระยะออกดอกติดฝัก 20 วันหลังงอก

| รายการ                                     | อัตราและวิธีตามต้นฉบับ                           |
| ------------------------------------------ | ------------------------------------------------ |
| flocoumafen 0.005%                         | 20 ก้อน/จุด จำนวน 10 จุด/ไร่                     |
| เหยื่อโปรโตซัว _Sarcocystis singaporensis_ | 3 ก้อน/จุด จำนวน 10 จุด/ไร่                      |
| ระยะห่าง                                   | 5-10 เมตร                                        |
| ตำแหน่ง                                    | ร่องรอยหนู ทางวิ่ง รอบแปลงติดป่า คันนา และดงหญ้า |
| PHI                                        | ไม่ระบุ ต้องตรวจฉลากและข้อกำหนดปัจจุบัน          |

ต้นฉบับระบุว่าโปรแกรมนี้เพิ่มประสิทธิภาพต่อหนูสกุลพุก (_Bandicota_) และหนูสกุลท้องขาว (_Rattus_) แต่ไม่ได้เพิ่มประสิทธิภาพต่อหนูหริ่ง (_Mus_) และเพิ่มต้นทุน

## 3. ระยะออกดอกติดฝัก 50 วันหลังงอกและต้นทุน

ใช้ flocoumafen และเหยื่อโปรโตซัวในอัตราเดิม เมื่อเหยื่อพิษถูกกินให้เติมกลับเท่ากับจำนวนเริ่มต้น ส่วนเหยื่อโปรโตซัวให้คงไว้ 3 ก้อน/จุดตามต้นฉบับ

| รายการต้นทุนตามต้นฉบับ |    บาท/ไร่ |
| ---------------------- | ---------: |
| Zinc phosphide         |      35.00 |
| Flocoumafen 0.005%     |     665.60 |
| เหยื่อโปรโตซัว         |     120.00 |
| **รวม**                | **820.00** |

### สรุปลำดับใช้

1. ระยะเตรียมดิน: zinc phosphide ออกฤทธิ์เร็ว 1 ครั้ง
2. 20 วันหลังงอก: flocoumafen ออกฤทธิ์ช้าร่วมกับเหยื่อโปรโตซัว
3. 50 วันหลังงอก: flocoumafen ออกฤทธิ์ช้าร่วมกับเหยื่อโปรโตซัว

## จุดที่ต้องระวังและตรวจสอบ

- คำเตือนระดับสูง: สารกำจัดหนูและเหยื่อพิษอาจทำให้คน เด็ก สัตว์เลี้ยง สัตว์ป่า และสัตว์ที่ไม่ใช่เป้าหมายเสียชีวิต ต้องใช้เฉพาะผลิตภัณฑ์ที่ได้รับอนุญาต วางในภาชนะหรือสถานีเหยื่อที่ป้องกันการเข้าถึง เก็บกู้เหยื่อและซากสัตว์อย่างปลอดภัย และปฏิบัติตามฉลากอย่างเคร่งครัด
- ระยะ 20 วันหลังงอกถูกเรียกว่า “ช่วงระยะออกดอก ติดฝัก” ในต้นฉบับ ซึ่งอาจไม่ตรงกับพัฒนาการของถั่วเหลืองทุกพันธุ์ ต้องยึดอายุวันตามโปรแกรมและตรวจงานวิจัยต้นทาง
- ห้ามเพิ่มอัตรา เปลี่ยนความเข้มข้น หรือวางเหยื่อในจุดที่คนและสัตว์ที่ไม่ใช่เป้าหมายเข้าถึง
- ต้องตรวจทะเบียน ฉลากปัจจุบัน วิธีเก็บกู้เหยื่อ ซากหนู และมาตรการลดพิษทุติยภูมิ
- ผลต่อหนูแตกต่างตามสกุล และต้นฉบับระบุว่าโปรแกรมร่วมไม่ได้เพิ่มประสิทธิภาพต่อหนูหริ่ง (_Mus_)
- ต้นทุนเป็นข้อมูล ณ เวลาทำวิจัย ไม่ใช่ราคาปัจจุบัน
- ต้นฉบับไม่ระบุ PHI ต้องตรวจฉลากและข้อกำหนดผลผลิตก่อนใช้
- ข้อมูลนี้ใช้เพื่อการเรียนรู้และค้นข้อมูล ไม่ใช้แทนทะเบียน ฉลากผลิตภัณฑ์ หรือข้อกำหนดกฎหมายปัจจุบัน ก่อนใช้จริงต้องตรวจสอบผลิตภัณฑ์ พืชเป้าหมาย สูตร อัตรา วิธีใช้ ลำดับการใช้ ความเข้ากันได้ ระยะเว้นก่อนเก็บเกี่ยว และข้อจำกัดบนฉลากทุกครั้ง
