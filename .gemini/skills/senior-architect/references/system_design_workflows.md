# System Design Workflows

Step-by-step guides for common system design tasks.

## 1. Capacity Planning
How to calculate required resources for a system.

1. **Estimate traffic:** DAU, peak QPS
2. **Estimate data size:** Row size, retention period
3. **Calculate storage:** Size * Retention
4. **Calculate bandwidth:** Request size * QPS
5. **Determine redundancy:** N+1 or N+2
