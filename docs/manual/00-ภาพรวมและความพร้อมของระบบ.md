# ภาพรวมและความพร้อมของระบบ

## ระบบนี้คืออะไร

NPT Smart Agri Dashboard เป็นระบบเว็บสำหรับใช้งานข้อมูลเกษตรจังหวัดนครปฐมจริง ไม่ใช่หน้าสาธิตแยกต่างหาก ระบบรวมข้อมูลจากหลายกลุ่มงานไว้ในฐานข้อมูลกลาง และให้ผู้ใช้แต่ละบทบาทเปิดดู ค้นหา วิเคราะห์ นำเสนอ และดูแลข้อมูลผ่านเว็บเดียวกัน

## งานที่ระบบรองรับ

- ผู้บริหารดูภาพรวมจังหวัดและสถานการณ์สำคัญ
- เจ้าหน้าที่เปิดดูข้อมูลตามกลุ่มงานและภารกิจ
- ผู้ดูแลระบบจัดการสิทธิ์ผู้ใช้และตรวจประวัติการใช้งาน
- ประชาชนหรือ guest เปิดดูข้อมูลสาธารณะที่เผยแพร่ได้
- ทีมพัฒนาและจังหวัดอื่นใช้เอกสารติดตั้งเพื่อขยายผลต่อ

## โครงสร้างการทำงานโดยย่อ

1. รวบรวมข้อมูลจากแหล่งงานของสำนักงานเกษตรจังหวัด
2. ตรวจรูปแบบและทำความสะอาดข้อมูล
3. นำเข้าฐานข้อมูล Supabase
4. ควบคุมสิทธิ์ด้วย role และ RLS
5. แสดงผลผ่าน Dashboard, Search, Map และ AI
6. ผู้ดูแลระบบติดตามการใช้งานและปรับปรุงข้อมูลต่อเนื่อง

## การควบคุมสิทธิ์และข้อมูล

- แยกสิทธิ์ guest, viewer, editor และ admin
- ใช้ Row Level Security ใน Supabase สำหรับข้อมูลที่ต้องควบคุม
- มีแนวทางป้องกันข้อมูลส่วนบุคคลก่อนเผยแพร่
- มี audit log สำหรับงานสำคัญของผู้ดูแลระบบ

## หลักฐานว่าระบบพร้อมใช้งานจริง

- มีโครงสร้างฐานข้อมูลและเอกสาร schema
- มีคู่มือการติดตั้งและ deploy
- มีคู่มือดูแลระบบหลังส่งมอบ
- มีคู่มือข้อมูลสำหรับการรวบรวม ทำความสะอาด และนำเข้า
- มีหน้าเว็บคู่มือให้เจ้าหน้าที่และทีมพัฒนาอ่านต่อได้

## Checklist สำหรับตรวจความพร้อม

- [ ] ระบบมีหน้าใช้งานจริง ไม่ใช่ mockup
- [ ] ข้อมูลหลักแสดงผลจากฐานข้อมูลหรือแหล่งข้อมูลที่เชื่อมไว้
- [ ] ผู้ใช้แต่ละบทบาทเข้าถึงข้อมูลไม่เท่ากันตามสิทธิ์
- [ ] มีคู่มือสำหรับเจ้าหน้าที่ ผู้ดูแลระบบ และทีมพัฒนา
- [ ] มีแนวทางดูแลหลังส่งมอบ เช่น deploy, backup, audit และ training
- [ ] มีเอกสารอ้างอิงที่ช่วยให้จังหวัดอื่นนำไปปรับใช้ต่อได้
