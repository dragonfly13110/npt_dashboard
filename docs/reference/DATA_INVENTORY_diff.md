## 🌐 ข้อมูลจาก API ภายนอก (Real-time)

**ไม่เก็บลงฐานข้อมูล - แสดงผลแบบ Real-time ผ่าน Netlify Functions Proxy**

| API                        | ข้อมูลที่ได้                                                 | ความถี่อัปเดต |
| -------------------------- | ------------------------------------------------------------ | ------------- |
| **Open-Meteo Weather**     | สภาพอากาศปัจจุบัน, พยากรณ์ 5 วัน, อุณหภูมิ, ความชื้น, ลม     | Real-time     |
| **Open-Meteo Air Quality** | AQI, PM2.5, PM10, คุณภาพอากาศ                                | Real-time     |
| **MOC Prices**             | ราคาสินค้าเกษตรจากกรมการค้าภายใน                             | รายวัน        |
| **GISTDA VIIRS**           | จุดความร้อนจากดาวเทียม                                       | รายวัน        |
| **RID Reservoir**          | ระดับน้ำในเขื่อน/อ่างเก็บน้ำ                                 | รายวัน        |
| **DOAE/RSS News**          | ข่าวสารจาก 6+ แหล่ง (เกษตรจังหวัด, กรมฯ, ESC, ICTC, Agritec) | Real-time     |
| **KKU Chatbot API**        | แชทบอทตอบคำถามเกษตรสาธารณะ                                   | On-demand     |
