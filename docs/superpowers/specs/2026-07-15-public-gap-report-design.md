# รายงานพื้นที่รับรอง GAP สาธารณะ

## เป้าหมาย

เพิ่มหน้า `/public/certifications` สำหรับดูรายงานมาตรฐาน GAP จังหวัดนครปฐม โดยใช้หน้ามาตรฐาน GAP เดิมในโหมดสาธารณะ เพื่อให้กราฟ ตัวกรอง และตารางมีข้อมูลชุดเดียวกันโดยไม่ทำสำเนาโค้ด

## หน้าจอและข้อมูล

- แสดงตัวกรอง: ปีรับรอง, ปีหมดอายุ, อำเภอ, ตำบล และชนิดพืช
- แสดงกราฟเดิม: จำนวนแปลงตามชนิดพืช, จำนวนแปลงตามอำเภอ, ปริมาณผลผลิตรวม และแนวโน้มวันหมดอายุใบรับรอง
- แสดงตารางรายแปลง: ชนิดพืช, ประเภทแปลง, พื้นที่, ปริมาณผลผลิต, วันรับรอง, วันหมดอายุ, หมู่/ตำบล/อำเภอของแปลงและเกษตรกร
- การ์ด “พื้นที่รับรอง GAP” บนหน้าแรกเชื่อมไปยัง `/public/certifications`

## ความเป็นส่วนตัว

API `/api/public-certifications` จะไม่เลือกหรือส่งชื่อ-นามสกุล, เบอร์โทรศัพท์ และรหัสแปลง ข้อมูลเขตการปกครองระดับหมู่/ตำบล/อำเภอยังคงเผยแพร่ได้

ในโหมดสาธารณะ กราฟที่เดิมอ้าง “จำนวนเกษตรกร” จะแสดงเป็น “จำนวนแปลง” และนับจากระเบียนที่ได้รับจาก API โดยตรง

## การทดสอบ

เพิ่มการทดสอบที่ตรวจว่าเส้นทางสาธารณะใช้โหมดสาธารณะ และการตอบกลับจาก API ไม่มีฟิลด์ระบุตัวบุคคลหรือรหัสแปลง
