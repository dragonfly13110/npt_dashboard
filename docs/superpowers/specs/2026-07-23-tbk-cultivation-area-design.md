# พื้นที่เพาะปลูกตาม ทบก.

## เป้าหมาย

เพิ่มหน้า `พื้นที่เพาะปลูกตาม ทบก.` ในกลุ่มยุทธศาสตร์และสารสนเทศ โดยแสดงข้อมูลครบทุกประเภทจากรายงาน DOAE ปีปัจจุบัน จังหวัดนครปฐม ตามค่าตัวเลือก:

- ทุกอำเภอและตำบล
- ทุกกลุ่มพืช ชนิด และพันธุ์
- รวมพืช ปศุสัตว์ นาเกลือ เพาะเลี้ยงสัตว์น้ำ และแมลงเศรษฐกิจ

## แหล่งข้อมูล

`https://farmer.doae.go.th/plants_detail/plants_select/report_select`

ตารางต้นทางมี 10 คอลัมน์:

1. รหัส
2. จังหวัด/อำเภอ/ตำบล/หมู่
3. พืช/พันธุ์พืช
4. จำนวนครัวเรือน
5. จำนวนแปลง
6. เนื้อที่ (ไร่)
7. ครัวเรือนประสบภัย
8. แปลงประสบภัย
9. เนื้อที่ประสบภัย (ไร่)
10. เนื้อที่คงเหลือ (ไร่)

## การจัดเก็บและอัปเดต

- ใช้รูปแบบล็อกอินและดึง HTML เดียวกับ `scrape_farmer_registry.js`
- แปลงตัวเลขที่มี comma เป็นค่าตัวเลขก่อนบันทึก
- เก็บ snapshot แยกตามปีการผลิตและเวลาที่ดึงข้อมูล
- ตรวจจำนวนแถวและโครงสร้าง 10 คอลัมน์ก่อนเขียนฐานข้อมูล เพื่อไม่ให้ข้อมูลเดิมเสียเมื่อหน้า DOAE เปลี่ยน
- อัปเดตอัตโนมัติเดือนละ 2 ครั้ง และรองรับการสั่ง sync โดยผู้ดูแล

## หน้าแสดงผล

- เพิ่มเมนูในกลุ่มยุทธศาสตร์และสารสนเทศ
- การ์ดสรุปจำนวนรายการ จำนวนแปลง เนื้อที่ ภัยธรรมชาติ และเนื้อที่คงเหลือ
- ระบุชัดว่าผลรวมครัวเรือนเป็นผลรวมตามรายการพืช/พันธุ์ ซึ่งอาจนับครัวเรือนซ้ำ
- ตารางค้นหาและกรองประเภทข้อมูล/ชนิดหรือพันธุ์
- ส่งออกข้อมูลที่กรองแล้วเป็น Excel
- แสดงปีข้อมูล วันตัดยอด และเวลาที่ระบบอัปเดต

## ขอบเขต

รอบแรกเก็บข้อมูลระดับจังหวัดตามตัวเลือกในภาพ ไม่ยิงรายงานแยก 7 อำเภอ เพราะต้นทางที่อนุมัติเป็นผลรวมจังหวัด หากต้องการมิติรายอำเภอค่อยเพิ่มเป็นงานแยกภายหลัง

## การตรวจสอบ

- parser test ด้วยแถวตัวอย่างและตัวเลข comma/ทศนิยม
- sync test ว่าไม่เขียนฐานข้อมูลเมื่อ HTML ไม่ครบ
- page test สำหรับการคำนวณยอดตามตัวกรองและ Excel
- ตรวจ build และชุดทดสอบเดิมก่อนส่งมอบ
